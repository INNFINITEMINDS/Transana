Creating Keyword Groups and Keywords

Transana allows you to select a preferred, or default, keyword group as part of defining an episode record. It is easier to describe the process of setting up a video for use in Transana if you create some keywords first. Once you are using Transana for your own work, it's likely that you will create your keyword groups and keywords later in the analysis process and assign them after you have started your analysis, but the process is easier to explain if you create some keywords beforehand. You use the Data window to create and manage keywords.

Activity: Follow these directions (Macintosh single-button mouse users should Control-click instead of right-click):

1. To create a keyword group, right-click the Keywords node in the database tree, then click Add Keyword Group.

You see the following dialog box:

2. Enter Demonstration Keywords and click OK. Transana creates your first keyword group.

(Please note that this keyword group will not be saved in the Transana database until you put some keywords in it! If you quit Transana right now, that group will not exist when you restart the program.)

3. The next step is to add a keyword to this group. Right-click Demonstration Keywords, then click Add Keyword from the resulting menu. Type in the keyword Geometry and an optional definition, then click OK.

4. Now add the keywords Flip, Quilt, and Symmetry. Note that each keyword is added to the database tree as you enter it.
