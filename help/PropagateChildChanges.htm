Propagate Quote and Clip Changes

As you work with your data in Transana, you may find that you end up with multiple copies of some Quotes and Clips. You may copy items because they fit into several places in your analytic structure, or you may create copies of Quote and Clips as a result of converting Search results to Collections. You may find that you want to make identical changes to all copies of a particular Quote or Clip. The "Propagate Quote and Clip Changes" function allows a researcher to synchronize copies of Quotes and Clips if desired.

(Please see Propagate Document and Episode Changes for a mechanism for keeping your Quotes and Clip Transcripts synchronized with your source Documents and Episode Transcript. The techniques discussed here do not affect the source Document or Episode Transcript.)

If you only need to update a Quote's text or the Transcript of a Clip, you can load one copy of the Quote or Clip into Transana's main interface, edit the text, and press the  button in the Document Window's toolbar. Transana will locate all Quotes taken from the same Document or all Clips taken from the same Episode Transcript with the same Quote or Clip ID and the same start and end points, and will change the appropriate text to match.

If you are interested in making broader changes to all copies of a Quote or Clip, you can load the Quote in the Quote Properties form or the Clip in the Clip Properties form. You can then change the item's ID, text, and keywords. Then press the  button on the dialog box. Transana will locate items with the original ID and the same start and end points that were created from the same source Document or Episode Transcript, and it will alter them so that the ID, text, and keywords match the current item. Please note that this function does NOT alter the original Document or Episode Transcript, and it does not affect copies of the item that have different IDs, nor will it remove keywords that designate a Clip as Keyword Examples.

The "Review Changes" screen will be displayed for each copy of the item Transana is able to identify. Press "Update" to replace the ID, text, and keywords and then move on to the next item copy. "Update All" will apply the changes to all copies of the item. (Use this button with caution!) "Skip" will prevent changes from being made to the current item, while "Cancel" will stop the change propagation process and revert all Quote or Clip copies back to their original form. The item you edited, however, will retain your changes.

A variety of errors can occur during the change propagation process, and the results may differ from what you might expect. Please take a moment to review the "Change Propagation" report to ensure that all item copies were correctly updated. For example, if a Quote or a Clip Transcript is locked by another user in multi-user Transana, that item will not be updated. You will need to wait until the other user releases the record lock, then repeat the change propagation or edit that item individually.
