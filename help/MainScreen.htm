Main Screen

When you start Transana, you will see the Transana main interface, made up of a menu bar and four windows: the data window, the document window, the media window, and the visualization window. The snapshot window and report windows are also available when needed.

The data window is where you add, organize, and manipulate data objects in Transana. The data window is where you select which documents, media files, and still images you want to work with in other windows, and the data window is where you initiate many analytic tasks in Transana.

The document window is where you display and edit text documents and transcripts of media files.

The media window is where you control playback and positioning of audio and video files and where video is displayed. When working with audio files, this window displays only the playback controller bar.

The visualization window displays visual representations of the data loaded in the Transana interface in the form of a waveform diagram representing the audio track of media data or of a bar chart showing coding that has been applied to text and media files during the analytic process.

The snapshot window displays still images and provides tools for manipulating and coding these images during the analytic process.
