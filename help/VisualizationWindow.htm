Visualization Window

Transana's Visualization window shows visual representations of information about your data. You can interact with your data in a number of ways in this window, and you can zoom in and out of the visualization to see detail or overview as you desire.

For Documents and Quotes, the Visualization Window displays coding across the length of the document. You can click in the visualization to place your document cursor, including selecting a segment of the visualization matching a coding bar to see the segment of the document that bears the coding shown. The Data window's Selected Items tab allows you to determine what Quotes are associated with the coding at a particular position in this keyword visualization. Hold your mouse over a colored band to see which Keyword the band represents.

For Episode Transcripts and Clips, there are several different visualizations to choose from. You can select the visualization you would like to use in the "Options" menu under "Media Visualization Style".

The waveform visualization displays a waveform, a visual representation of the intensity of the audio track for a given media file. This visual representation can be extremely useful during transcription for precise navigation within a video or audio file. The waveform also facilitates the process of inserting time codes accurately.

The keyword visualization shows you the coding you have applied to your media across the media file's time line. This can be very useful during analysis to see how you have applied keywords to your Episode as a whole. The Data window's Selected Items tab allows you to determine what Clips are associated with the coding at a particular position in the keyword visualization. Hold your mouse over a colored band to see which Keyword the band represents.

The hybrid visualization shows both waveform and keyword information for media-based data.

The keyword and hybrid visualizations can be customized. Click the Filter dialog button () to load the visualization Filter dialog box. In this dialog, you can specify which Quotes, Clips, Snapshots, and Keywords to include or exclude from the visualization, as well as the color associated with each Keyword. You can also customize the size of the bars and the amount of whitespace that appears between bars.

Clicking on a spot in the visualization window will position the Document cursor in a Document that is not in Edit mode, and will position the cursor in a Transcript not in Edit mode as well as positioning the media file to the corresponding location. The Visualization window's status bar gives you precise information about the position of the Document cursor or media cursor in the media file.

You can zoom in and out on the visualization using the zoom buttons with the magnifying glass icons in the lower left-hand corner of the Visualization window. To zoom in, highlight the portion of the visualization you would like to examine more closely and press the  button. If you zoom in several levels in this way, the  button will back you out in the same stages you zoomed in with. You can also press the "100%" marker at any time to return the visualization to its original size.

When working with media-based data, you can alter your current position in the media file very precisely by pressing the cursor right and cursor left keys within the visualization. The amount the media moves depends on several things. If you press the cursor left or cursor right key, the visualization cursor moves one pixel in the direction indicated. How much time this represents in the media file depends on how long your media file is, how large the visualization window is on your screen, and whether or not you have zoomed in on a portion of the visualization.

Another alternative is to hold down a modifier key (Shift, Alt, or Ctrl/Cmd) when using the cursor key, which causes the media file to shift a specific time amount. If you are holding the Shift key when you press the cursor key, the media file moves in one-second increments. If you are holding the Alt key, the media file moves in 1/2-second increments. If you hold the Ctrl key on Windows and the Command key on OS X, the media file moves 1/29.97th of a second, or exactly one frame in a standard NTSC video file.

Right-clicking the Visualization window causes the media file to play or pause, starting from the selected position. You can drag in the Visualization window with your mouse to select a segment of the media file, to set both a start and stop point for playback. (Alternately, you can click in the waveform to indicate a starting point and hold down the Shift key while clicking a second time to indicate an ending point to make a selection.)

You can create Clips without Transcripts in Transana using the "create transcript-less clip"   button in the Visualization window. First, make a precise selection within the Visualization window. Then, to create a transcript-less Standard Clip, select the destination Collection in the data tree and press this button. To create a transcript-less Quick Clip, select one or more Keywords in the data tree and press this button.

The looping button  allows you to play a section of your media file in a loop, which can be very helpful during transcription. You can make a selection in the Visualization window (zooming in for greater precision) and that selection will loop when you press the looping button. If you do not make a specific selection, a 5 second segment starting at the current media position will loop. Pressing the looping button again is the only way to stop the looping process. Other methods of stopping media playback do not work when looping is in effect.
