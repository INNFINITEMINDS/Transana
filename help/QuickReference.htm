Quick Reference

Add Clip Highlight the section of your transcript you wish to turn into a clip. Windows and Mac alternative: right-click the Collection where you want the Clip and select "Add Clip" from the popup menu. Windows only alternative: you can also simply drag the selected text onto a Collection in the data window. NOTE: You must have time codes in your transcript for this to work correctly. Regardless of what you highlight, the video portion of the clip will extend from the last time code preceding your highlight to the first time code following it.
Add a Quick Clip Highlight the section of your transcript you wish to turn into a clip. Windows and Mac alternative: right-click the Keyword you want for the Quick Clip and select "Create Quick Clip" from the popup menu. Windows only alternative: you can also simply drag the selected text onto a Keyword in the data window. NOTE: You must have time codes in your transcript for this to work correctly. Regardless of what you highlight, the video portion of the quick clip will extend from the last time code preceding your highlight to the first time code following it.
Add Clip Note Right click on the clip you wish to add a note to in the data window and select "Add Note."
Add Collection Right click on the Collection Root Node in the data window, and select "Add a Collection."
Add Collection Note Right click on the collection you wish to add a note to in the data window, and select "Add Note."
Add Episode In the data window, right click on the series you wish to include the episode in, and select "Add Episode." Adding an Episode will automatically prompt you to create a Transcript for that Episode. NOTE: An Episode must be placed within a Series.
Add Episode Note Right click on the Episode you wish to attach a note to in the data window, and select "Add Note."
Add Keyword Group Right-Click on the Keywords Root Node and select "Add Keyword Group." You can also create Keyword Groups via the Keyword Management dialog box. Please note that Transana will not save an empty Keyword Group.
Add Keyword to All Clips in a Collection Windows and Mac alternative: right-click the Keyword and select "Copy," then right-click a Collection and select "Paste." Windows only alternative: drag the Keyword onto the Collection in the Data window.
Add Keyword to All Episodes in a Series Windows and Mac alternative: right-click the Keyword and select "Copy," then right-click the Series and select "Paste." Windows only alternative: drag the Keyword onto the Series in the Data window.
Add Keyword to Clip There are several ways to do this. Windows and Mac alternatives: right-click the Keyword and select "Copy," then right-click the Clip and select "Paste;" double-click the appropriate keyword in the Add Clip/Clip Properties window; select the Keywords tab in the Data window and right-click to select the "Edit" option; or press the "Edit Keywords" button on the Transcript tool bar. Windows only alternatives: drag the Keyword onto the Clip in the Data window; drag the Keyword from the Data window onto the Clip Transcript in the Transcript window.
Add Keyword to Episode There are several ways to do this. Windows and Mac alternatives: right-click the Keyword and select "Copy," then right-click the Episode and select "Paste;" double-click the appropriate keyword in the Add Episode / Episode Properties window; select the Keywords tab in the Data window and right-click to select the "Edit" option; or press the "Edit Keywords" button on the Transcript tool bar. Windows only alternatives: drag the Keyword onto the Episode in the Data window; drag the Keyword from the Data window onto the Episode Transcript in the Transcript window.
Add Keyword to Keyword Group Right-click on the keyword group you wish to add a keyword to. Select "Add Keyword." Keywords can also be added in the Keyword Management dialog.
Add Nested Collection Right-click a Collection and select "Add Nested Collection."
Add Series Right-click the Series Root Node and select "Add Series"
Add Series Note Right-click the Series and select "Add Note"
Add a Transcript Right-click on the Episode you wish to add a transcript to. Select "Add Transcript." Adding a Transcript will automatically load that transcript for editing and playback.
Add Transcript Note Right click on the Transcript object in the Data window that you wish to add a note to and select "Add Note."
Change Visualizations Go to the "Options" menu, select "Visualization Style", and select the visualization you would like to use.
Clear All Search Results Right-click the Search Results Root Node and select "Clear All." NOTE: This option is only available when searches appear in the Search Results list.
Clear Search Result Right-click the Search Result and select "Clear"
Convert Search Results to a Collection Right-click the Search Result and select "Convert to Collection."
Copy all Clips from a Collection to a New Collection Windows and Mac alternative: right-click the source Collection on the Database tab in the Data window and select "Copy," then right-click the target Collection and select "Paste.: Windows only alternative: select the originating Collection on the Database tab in the Data window and drag it to a new Collection name. Be sure to hold down the Ctrl key to Copy rather than Move the Clips.
Copy Clip to New Collection Windows and Mac alternative: right-click the Clip and select "Copy," then right-click the target Collection and select "Paste." Windows only alternative: Select the Clip and drag it to the target Collection. Be sure to hold down the Ctrl key to Copy rather than Move the Clip.
Copy Keyword to a new Keyword Group Windows and Mac alternative: Right-click the Keyword and choose "Copy," then right-click the new Keyword Group and choose "Paste." Windows only alternative: Select the Keyword and drag it onto the new Keyword Group. Be sure to hold down the Ctrl key to Copy rather than Move the Keyword.
Define a Clip as a Keyword Example Windows and Mac alternative: right-click the desired clip and select "Copy," then right-click the Keyword and select "Paste." Windows only alternative: From the Data Window, drag the desired clip onto the desired Keyword.
Delete Clip Right-click a Clip and choose "Delete." NOTE: All associated Clip Keywords will also be deleted.
Delete Collection Right click on the Collection you wish to remove and select "Delete." NOTE: All associated Clips will also be deleted.
Delete Collection Note Right click on the Note you wish to remove and select "Delete."
Delete Episode Right click on the Episode you wish to delete, and select "Delete." NOTE: All associated Transcripts and Episode Keywords will also be deleted.
Delete Episode Note Right click on the note you wish to remove, and select "Delete."
Delete Keyword from Clip Right-click the Clip, and select "Properties." Then highlight the Keyword and click the "<<" button. Click OK on the Clip Properties form.
Delete Keyword from Episode Right-click the Episode, and select "Properties." Then highlight the Keyword and click the "<<" button. Click OK on the Episode Properties form.
Delete Keyword from Keyword Group Right click on the keyword you wish to remove and select "Delete." NOTE: Deleting a Keyword removes it from all Episodes and Clips to which it has been applied.
Delete Keyword Group Right click on the Keyword Group you wish to remove and select "Delete." NOTE: All associated Keywords will also be deleted, and will be removed from all Episodes and Clips.
Delete Note (any type) Right-click the Note entry in the Database tab in the Data window and select "Delete."
Delete Series Right click on the Series you wish to remove, and select "Delete." NOTE: All related Episodes, Episode Keywords, Transcripts, and Notes will also be deleted.
Delete Series Note Right click on the note you wish you remove, and select "Delete."
Delete Transcript Right-click the Transcript on the Database Tab of the Data window and select "Delete."
Edit Clip Note Double-click the Note in the Database Tree.
Edit Clip Properties Right-click on the Clip in the Database Tree and select "Properties."
Edit Collection Properties Right click on the Collection in the Database Tree and select "Properties."
Edit Collection Note Double-click the Note in the Database Tree.
Edit Episode Note Double-click the Note in the Database Tree.
Edit Episode Properties Right click on the Episode in the Database Tree and select "Properties."
Edit Note Properties (any type) Right click on the Note in the Database Tree and select "Properties."
Edit Series Properties Right click the Series in the Database Tree and select "Properties."
Edit Series Note Double-click the Note in the Database Tree.
Edit Transcript Load the appropriate Transcript by double-clicking the Transcript of an Episode or by double-clicking a Clip. Put the Transcript into "Edit mode" by clicking on the icon in the Transcript toolbar.
Edit Transcript Properties Right click the Transcript in the Database Tree and select "Properties."
Export Clip Data to a tab-delimited text file Right-click an Episode or a Collection. Select "Clip Data Export". Select the name (and location) for the exported data file, and use the Filter Dialog to determine what data the file should include.
Insert a time code into a Transcript Position the Transcript cursor in the desired spot in the Transcript within the Transcript window. Then get the video to the proper position by playing the video and either by pressing Pause (not Stop) in the Video window or by right-clicking the Wave Form in the Visualization window. Then click "Current" in the Visualization window or press Ctrl-T from within the Transcript.
Keyword Management Right-click the Keywords Root Node and select "Keyword Management." Or click the icon on the Episode Properties, Clip Properties, or Edit Keywords screens.
Load a Video First create or select a Series, then add an Episode to it, then add a Transcript. Loading a Transcript is the action that loads a video into the Video window.
Locate a Clip in the original Episode Right-click the Clip and select "Locate Clip in Episode."
Merge Keywords Right-click a Keyword and rename the Keyword Group and Keyword to match another existing keyword.
Move all Clips from a Collection to a New Collection Windows and Mac alternative: right-click the source Collection and select "Cut," then right-click the target Collection and select "Paste." Windows only alternative: Select the source Collection and drag it to the target Collection name. Do not hold down the Ctrl key to Move rather than Copy the Clips.
Move Clip to New Collection Windows and Mac alternative: right-click the Clip and select "Cut," then right-click the target Collection and select "Paste." Windows only alternative: Select the Clip and drag it to a new Collection name. Do not hold down the Ctrl key to Move rather than Copy the Clip.
Move Keyword to a new Keyword Group Windows and Mac alternative: right-click the Keyword and select "Cut," then right-click the target Keyword Group and select "Paste." Windows only alternative: Select the Keyword and drag it onto the new Keyword Group. Do not hold down the Ctrl key to Move rather than Copy the Keyword.
Open Clip Double-click the Clip within a Collection in the Data window.
Open Episode To view a video file in an Episode, you must open a Transcript for that Episode. If one does not exist, add a Transcript.
Open Note Double-click the Note in the Data window.
Open Transcript Double-click the Transcript within the Data window.
Play all Clips in a Collection Right-click the Collection on the Database Tab in the Data window and select "Play All Clips."
Play from a position to the end of the Episode/Clip Right-click in the Visualization Window. Windows only alternative: Ctrl-right-click in the Transcript window. Mac only alternative: hold down the "Command" or "Meta" key and click in the Transcript window.
Print a Transcript Open the Transcript you wish to print. From the File Menu select Print Transcript or from the Transcript menu, select Print.
Re-order Clips within a Collection. Windows and Mac alternative: Use cut-and-paste to change the order of Clips within a Collection. When you paste a Clip, it will be positioned before the Clip you paste it on. Windows only alternative: Use drag-and-drop to change the order of Clips within a Collection. When you drag a Clip, it will be positioned before the Clip you drop it on.
Search Right-click the Search Results Root node. Define your Search and press OK. Then look at the resulting Search Result.
View Collection Summary Report Right-click a Collection name in the Data window and select "Collection Summary Report."
View Keyword Definition Right-click a Keyword and select "Properties." Keyword Definitions can also be viewed in the Keyword Management screen or through the Keyword Summary Report.
View Keyword Map Right-click an Episode and select "Keyword Map" to see a graphical representation of where different keywords have been assigned to Clips created from the selected Episode.
View Keyword Summary Report Right-click the Keywords Root to produce a report of all Keywords in all Keyword Groups. Right-click a Keyword Group to produce a report of all Keywords in that group.
View Keyword Usage Report Right-click a Series and select "Keyword Usage Report" to produce a report of the Keywords assigned to all Episodes within that Series. Right-click an Episode and select "Keyword Usage Report" to produce a report of all Keywords assigned to Clips created from that Episode. Right-click a Collection and select "Keyword Usage Report" to produce a report of the Keywords assigned to all Clips within that Collection.
View Series Keyword Bar Graph Right-click a Series and select "Series Keyword Bar Graph".
View Series Keyword Percentage Graph Right-click a Series and select "Series Keyword Percentage Graph".
View Series Keyword Sequence Map Right-click a Series and select "Series Keyword Sequence Map". To switch between the multi-line and single-line versions of this map, go to the Filter Dialog's Options tab and check or uncheck the "Single-line display" option.
