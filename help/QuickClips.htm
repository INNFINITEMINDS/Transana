Quick Clips

We now have a collection of standard clips that show what our sample video has to show about these students and their increasing understanding of the concept of symmetry. We notice that this teacher uses questions in some very interesting ways. For example, she never defines the term symmetry, but instead asks questions of the students in such a way that they develop a communal definition that appears to be understood and retained by more of the students than other teachers using more traditional approaches.

Transana provides an alternative method of creating clips called quick clips. Quick clips are very easy to create, and are designed to be created quickly. They are also explicitly limited to a flat (non-hierarchical) structure. This method of creating clips is appropriate for the sort of open/emergent coding suggested by grounded theory approaches, as well as for analytic approaches that focus on the analysis of Transana's maps and graphs rather than making use of the manipulation of clips in collections for extracting meaning from your media files.

Let's explore this teacher's use of questions using quick clips.

Activity: Follow these directions to create a quick clip of one of the teacher questions.

As mentioned earlier, quick clips are keyword-oriented clips, while standard clips are collection-oriented clips. In the same way that we had to create a collection before we could create a standard clip, we need to create a keyword before we can create a quick clip.

1. Right-click the Tutorial keyword group and choose Add Keyword. Create a keyword called Teacher Question and give it a definition, then press OK to complete the process.

2. If the Demo episode's Transcript transcript is not currently loaded, load it by double-clicking it in the data window's database tree.

3. At approximately 1:30.8, the teacher asks a question of the class about the software they are using. You can use the Visualization window to move quickly to that spot, or you can use the text search box in the transcript toolbar to search for the first reference in the transcript to the word "software".

4. Make sure that time codes are visible in your transcript, then use the mouse to click and drag the cursor to highlight the sentence about why their computer software won't let them do diagonal flips. The selection runs from about 1:30.8 to about 1:47.0 in the media file.

5. Right-click option: Right-click the Teacher Question keyword in the data window's database tree. Select the Create Quick Clip option.

Drag-and-drop option: Drag the selected text onto the Teacher Question keyword in the data window's database tree.

Another alternative: First check the Options menu, and make sure the Quick Clips Mode item is checked, indicating that quick clips mode is enabled. Drag a keyword from the data window's database tree onto the transcript. (If Quick Clips Mode is not checked, this action will add the keyword to the episode or clip currently loaded rather than creating a quick clip.)

Notice that Transana creates a collection called "Quick Clips" if one does not already exist. (Multi-user Version note: Transana automatically appends your user name onto the quick clips collection to differentiate between the quick clips created by different users.) You will also notice that Transana has created a clip named after the current episode and numbered it to differentiate it from other quick clips created from the same episode. If you load the quick clip (or look at the clip properties) you will also note that the quick clip has been assigned the keyword used to initiate the creation of the quick clip. (Multi-user Version note: Transana will also assign your user name to the quick clip as a keyword, creating a keyword group called "Transana Users" and adding your user name as a keyword to that group if necessary.)

If you want to create a series of quick clips with the same keyword, highlight the desired keyword in the data window's database tree, then for each selection in the transcript, you can press Ctrl-K or the Create Quick Clip button  to create the desired quick clips.

Activity: Create a number of quick clips very rapidly.

1. In the data window's database tree, select the Teacher Question keyword.

2. In the Transcript window, scroll to the top of the transcript and click on the top line.

3. In the Transcript Toolbar, type a question mark into the text search box.

4. Click the Search Forwards button in the Transcript Toolbar, the right arrow to the right of the text search box.

5. At each question mark, look at the question. Insert time codes at the beginning and end of each question, if necessary. (Remember, in the transcript, you can left-click to position the video in a time-coded segment, then right-click to play that segment.)

6. Then, if the question crosses time codes, highlight the question. If the questions does not cross time codes, just click in the question (so that there is no selection in the transcript.) If the question mark indicates unclear transcription or an incidental comment (such as "Okay?") that doesn't really constitute a meaningful question, search for the next question mark.

7. Press Ctrl-K, or click on the Create Quick Clip button   in the transcript toolbar to create the quick clip. Alternately, you can make a selection in the transcript and double-click the keyword in the database tree to create a quick clip. (It doesn't matter which method of creating a quick clip you choose. Try them all, then practice the one that feels most intuitive to you.)

8. Go back to step 4 above to repeat this process until you have created quick clips for all of the teacher questions in the demo video.

You should now have a collection of about 20 quick clips of teacher questions available for further exploration. If we wanted to, we could now rename the Quick Clips collection, calling it Teacher Questions or something equally clever. To do this, you can click on the collection name in the tree to make the collection name editable, or you can right-click the collection, choose Collection Properties, and rename the collection that way.

At this point, you may wish to create a collection note (or multiple clip notes if you prefer) covering the analytic memo data for the quick clips you just created.

Note: Because of how easy it is to create quick clips, some novice Transana users may be tempted to use quick clips in situations where they are not appropriate. A single clip with multiple keywords allows search-based hypothesis testing that is not possible with multiple identical clips, each with only a single keyword. Therefore, if you attempt to make multiple quick clips using the same selection and different keywords, Transana will apply the keywords to the same quick clip.
