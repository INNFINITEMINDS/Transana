Collection Properties

A Collection in Transana is a place to group together bits of analytic data, the small segments you select from Documents and Episodes, as well as Snapshots, that you find analytically important or interesting. One can think of a Collection like a pile of paper clippings from old-school manual methods, and it is also important to recognize that placing a Quote, Clip, or Snapshot in a Collection is one of the ways of coding data in Transana.

To add a Collection in Transana, right-click (Ctrl-click on the Mac) the "Collections" node in the Data window and choose "Add Collection." You will see the dialog box shown above. The only required information is a "Collection ID" that is unique in the parent object. "Owner" can be used to indicate the person who created or is in some other way responsible for the Collection, and "Comment" can be used in any way you wish. The "Default Keyword Group" is the Keyword Group that will be displayed by default when Quotes, Clips, or Snapshots are added to this Collection.

Collections can also be nested within other Collections. This allows Transana to represent more complex analytical structures and relationships within the Data Tree Display. Simply right-click (Ctrl-click on the Mac) an existing Collection and select the "Add Nested Collection" option.
