Starting Transana

You should run Transana while you do this tutorial so that you can perform the tasks as they are explained. If you are already running the tutorial from within Transana, skip ahead to the next page now.

Microsoft Windows Activity: To start Transana, click Start on the Taskbar, point to Programs, select the Transana 2 Program Group, then click Transana 2.0x from the submenu.

Macintosh Activity: To start Transana, open your Applications folder, double-click the Transana 2 folder, and then double-click the Transana program icon.

The single-user version of Transana prompts you to select a database.

The multi-user version of Transana requires that you log in to your database server, as well as select a database. Transana prompts you for your user name, password, database server, and database name. If you don't have a username, password, and database server name, ask your project administrator or your technical support representative for this information.

If you already have a tutorial database, select that from the list. If you don't already have a tutorial database, you can make up a database name, such as your initials followed by the word "Tutorial" and click OK. (Please note that spaces and punctuation symbols are not allowed in database names.) Transana creates the appropriate database files for you.
