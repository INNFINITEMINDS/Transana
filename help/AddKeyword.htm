Keyword Properties

To add a keyword in Transana, you need to define the Keyword Group to which the keyword belongs, the keyword itself, and, optionally, give a definition of the keyword. If you specify a Default Code Color, that color will be used in Transana's Maps, Graphs, and Snapshots by default, although you can change the color in individual Maps, Graphs, and Snapshots if you wish. You can also defind a default Code Tool, Line Width, and Line Style for use in Snapshot detail coding.

Although the Keyword Group field on this dialog box is a drop-down list which defaults to the Keyword Group value active when this dialog box is called, you can add a new Keyword Group at any time by typing over the displayed value. Changing the Keyword Group for an existing keyword will move that keyword to the new keyword group. All instances of that keyword will be updated automatically.
