Delete

Simply highlight one or more file names in the File Name window, then press the Delete button. The selected files will be deleted.

To delete a folder, you must first delete all the files from it. Then select it in the Folder tree and select the "Delete Folder" button.

For Experienced SRB users: Please be aware that the File Management Utility does not keep track of File Replication numbers, and is only able to delete the original copy of any given file.
