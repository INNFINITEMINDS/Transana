Export Data

To export data from a Transana database for transfer to another computer, go to the Tools menu and select "Export Data." You can choose to export your entire database or just your Codebook made up of all of your Keywords in Keyword Groups. Browse to a directory and enter the name you want to give to the data file. This file will have a ".tra" file extension to indicate that it is in Transana-XML format.

To transfer this data to another computer, you will need to copy the Transana-XML file created by the Export Database routine, as well as all related media files that the new computer does not already have access to, to the new computer. You will then need to run the "Import Data" routine on the new computer.

The Transana-XML file produced by this routine is intended only for use by Transana. It is generally not a good idea to try to edit this file unless you know exactly what you are doing.
