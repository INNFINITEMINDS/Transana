Data Window

Transana's Data Window is made up of as many as four tabs, depending upon what is currently loaded in Transana. The Database tab is always available, and displays the organizational structure of all Transana data in a tree format. The Episode Clips tab and the Selection Clips tab are only available when an Episode is loaded, and they display information about what Clips have been created from the loaded Episode. The Keywords tab is available when an Episode or a Clip is loaded in Transana, and displays information about the keywords that have been applied to the loaded Episode or Clip.

The Database tab helps you organize and manipulate your data. It is arranged in a tree structure, with the following possible nodes:

Series Root

     Series

         Episodes

             Transcripts

             Episode Notes

         Series Notes

Collections Root

     Collections

         Clips

             Clip Notes

         Collection Notes

         Nested Collections

             Clips within Nested Collections

Keywords Root

     Keyword Groups

         Keywords

             Keyword Example Collections

                 Keyword Example Clips

Search Results Root

     Search Result

         Series

             Episode

                 Transcript

         Collection

             Clip

             Nested Collections

                 Clips within Nested Collections

See Organizational Structure for more information on what these nodes mean.

Most elements in the database tab in the data window can be manipulated through their right-click menus, which vary depending on the type of object you select. (Macintosh users will find that holding down the "Ctrl" key on the keyboard while clicking emulates a right-click.) Typically, each element also has a default behavior which can be accessed by double-clicking the element. For Transcripts and Clips, this default behavior is to load and display the appropriate element in Transana, while for the remaining items the default is to Add the appropriate sub-object.

These right-click menus often include "Cut", "Copy" and "Paste" options for copying and moving data elements and manipulating them in other ways. You can copy or move Clips between Collections, add Keywords to Episodes and Clips, and copy or move Keywords between groups. Some cut, copy, and paste operations affect multiple objects. If you copy a Collection and paste it on another collection, all the Clips in the first Collection will be copied to the second. If you copy a Keyword onto a Collection, that Keyword will be applied to all the Clips in the Collection.

In the Windows version of Transana, all of the actions that can be done with cut, copy, and paste can also be accomplished with drag and drop. Dragging a node in the database tree onto another node is equivalent to cutting and pasting, while if you hold down the Ctrl key during the drop, that is the equivalent of copy and paste. Drag and drop is not supported on Macintosh OS/X at this time.

The Episode Clips tab in the Data Window displays all Clips that have been created from the loaded Episode in order of the Clip start time. You can load a Clip by double-clicking on it in this list.

The Selection Clips tab displays all Clips that have been created from the current location in the Episode's Transcript and Video. This list changes as the Episode plays, whenever you place the cursor in the Transcript, or whenever you select a location in the visualization or video windows. Please note that when a segment of the transcript is selected, only the clips related to the starting point of that selection are displayed until the video is played. You can load a Clip by double-clicking on it in this list.

The Keywords tab in the Data Window displays information about the keywords that have been assigned to the currently-loaded Episode or Clip. The keyword list can be edited by right-clicking within the list and selecting "Edit" from the popup menu, and keywords can be deleted by right-clicking them and selecting "Delete".
