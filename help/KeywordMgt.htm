Keyword Management

Keyword Management, available by right-clicking the Keywords root node on the Database tab of the Data window and from many of Transana's object properties forms, allows users to add Keyword Groups and Keywords quickly and easily. Please note that if you invoke the Keyword Management screen from within the Document Properties, Episode Properties, Quote Properties, Clip Properties, Snapshot Properties, or Edit Keywords screen, the ability to edit and delete Keywords will be disabled in order to prevent potential data corruption.
