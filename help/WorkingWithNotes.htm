Working With Notes

It is common in qualitative analysis to end up with a lot of analytic memos that contain a lot of information about the analysis you have performed. Transana provides the Notes Browser to make it easier for researchers to review their notes so that they can harvest insights and ideas from them during the writing of results and other times during the analytic process.

Activity: To access the Notes Browser, go to the Tools menu and select Notes Browser.

The notes browser organizes your notes by type, showing all of the collection notes as a group, all of the clips notes as a group, etc. This allows you to move through similar types of notes more quickly.

Activity: Click on a note to display the note text in the Notes Browser's right-hand pane.

You can edit the note there if desired. You can also right-click the note in the left-hand pane to get a popup menu of options for manipulating notes, including being able to locate notes in the database tree and open clips for review.

You can also search for text across notes to find notes that you are looking for more easily. For example, it is common to have ideas for further exploration of your data during your analytic work, ideas that you want to follow up on later so as not to interrupt your current train of thought. You might record such ideas in your notes and flag them with the phrase "Follow Up" to make them easier to find later.

Activity: Click on the Note Search tab in the Notes Browser. Type "Follow up" in the Search Text box, then press the Search button.

Only notes that contain the appropriate text will be included in the Notes window below the search box. To see all of your notes again, return to the Notes tab.

Activity: From the Notes tab, right-click any of the root nodes (Series, Episode, Transcript, Collection, or Clip.) Each node has its own Notes Report showing the notes under that that node. If you right-click the main Database node, you can get a Notes Report that includes all notes in your database. The Filter Dialog allows you to select a subset of notes to be included in the report if desired. The report can also be saved as a Rich Text Format document and can be printed.

It is also possible to including clip notes in Episode Report and to include clip and collection notes in the Collection Report by selecting that option through the filter dialog.
