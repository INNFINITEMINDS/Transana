Keyword Properties

A Keyword in Transana is an analytic tag that can be assigned to Quotes, Clips, and Snapshots to code these data objects. This coding is shown in Transana's reports and is accessible through the Search tool.

In Transana, all Keywords are stored in Keyword Groups, which allows for a two-level hierarchy in the coding system.

To create a Keyword in Transana, you need to define the Keyword Group to which the Keyword belongs, the Keyword itself, and, optionally, give a definition of the Keyword. If you specify a Default Code Color, that color will be used by default in Transana's Maps, Graphs, and Snapshots, although you can change the color in individual Maps, Graphs, and Snapshots if you wish. You can also defind a default Code Tool, Line Width, and Line Style for use in Snapshot detail coding.

Although the Keyword Group field on this dialog box is a drop-down list which defaults to the Keyword Group value active when this dialog box is called, you can add a new Keyword Group at any time by typing over the displayed value. Changing the Keyword Group for an existing keyword will move that Keyword to the new Keyword Group. All instances of that Keyword will be updated automatically.
