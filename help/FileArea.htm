The File Area

When you are working with local files, your File Area should look like this:

On the top, you can select the drive and folder you want to work with. On the bottom, you can select and filter files.

If you are connected to a SRB, your File Area should look like this:

Here, you can select Collections on the top and select and filter files on the bottom.

You can select more than one file by combining the Shift and Ctrl keys with mouse-clicking. Shift-click selects a range of files, while Ctrl-click highlights individual files to add to your list of selected files. The buttons in the center will act upon all selected files as appropriate.
