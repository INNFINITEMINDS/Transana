Keyword Summary Report

The Keyword Summary Report can be run from the Keywords root node in the data tree. In this case, it lists all keyword groups in your Transana database, and includes all keywords in each group with their definitions.

It can also be requested from any Keyword Group node in the Data Tree. In this case, it lists all keywords in the specified keyword group along with their definitions.

This report essentially presents all or part of your coding definitions. It is helpful for describing and sharing coding sets and in working on code definition consensus and inter-rater reliability.
