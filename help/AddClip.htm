Clip Properties

To add a Clip in Transana, you first highlight the section of the episode transcript you wish to turn into a clip. You then drag the selected text from the transcript onto a collection on the database tab of the data window. Alternately, once you have made your transcript selection, you can right-click (Ctrl-click on the Mac) on the collection where you want to place the clip and select "Add Clip" from the popup menu. A third option is to make your transcript selection and then double-click the target collection. Doing this will add the clip to the end of the list in the collection.

If you want to place the new clip in a particular place in the list of clips and snapshots in a collection, you can right-click an item in the list and choose "Add Clip," or drop the transcript text on a specific item in the list. If you do this, the new clip will be added above the selected item in the collection list.

Note that you must have time codes in your transcript for clip creation to work properly. The video displayed as the clip will start at the last time code preceding your highlighted transcript selection and end at the first time code following it. Only what you highlight will be included in the clip's transcript.

If you add a clip using any of the three techniques described above without first making a selection in the transcript, Transana will automatically use the text between the time codes that immediately precede and follow the current transcript cursor position.

You will then see the Add Clip dialog box.

The only required field is the "Clip ID," which must be unique within the parent collection. "Comment" can be used in any way you wish. By selecting a keyword group and keyword, then clicking the  button, you can add as many analytic keywords as you wish to each clip, enabling you to find this clip later when doing keyword searches. The  button allows you to delete keywords from the list, and the  button allows you to access Keyword Management. The "Collection ID," "Series ID," "Episode ID," "Clip Start," "Clip Stop," and "Clip Length" fields are all maintained by Transana and cannot be edited. While you can edit the clip text if you like, it is not recommended. Changes made here are not reflected in the original episode transcript.

When you edit a Clip using Clip Properties from the Clip's right-click menu, the  button will be visible. This button is used to locate copies that match the original Clip ID, source transcript number, clip start time, and clip end time, and to update those copies so that they are identical to the current clip. See Clip Change Propagation for more information.

One final note: Unlike other branches of the database tree, items within a collection have an "order" that can be altered by the user. (This is important for "Play All Clips" and in the Collection Report.) If you choose "Add Clip" or drop your transcript text on a collection name, your new clip will be added to the end of the list of clips and snapshots within the collection. However, if you choose "Add Clip" or drop your transcript text on a clip or snapshot, the new clip will be inserted in the list above the item where the text is dropped. Finally, the order of clips and snapshots within a collection can be altered using cut-and-paste or drag-and-drop within the collection.
