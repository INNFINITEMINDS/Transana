Other Transana Functions

Adjust Indices

Adjust Indices, located in the Document menu, allows you to alter all time codes in an editable Transcript by a constant amount. This is helpful when you want to use an existing time-coded Transcript for a new media file that may not start at precisely the same time.

Auto-Arrange

The Auto-Arrange option, in the Options menu, determines whether user changes in the size and position of one Transana window cause automatic changes in the size or position of the other Transana windows. If you have a dual-monitor setup, it can be very nice to turn Auto-Arrange off, move the Document Window or Media Window to your second monitor, and re-size and re-arrange your other Transana windows.

Auto Word-Tracking

The Auto Word-Tracking option is located in the Options menu. When it is checked, the Transcript will be highlighted as the media file plays. You must have time codes in your Transcript for this to work, and the Transcript must not be in "Edit" mode.

Bring Main Windows to Front

Located in the Window menu, the "Bring Main Windows to Front" item bring Transana's Document, Data, Media, and Visualization windows to the front, over any windows from Transana's Snapshots, Reports, or other applications.

Change/Create Database

Transana supports the creation and use of multiple, separate databases. To change from one database to another, select "Change Database" from the "File" Menu. If you want to create a new database, simply type in the name of a database that does not exist, and Transana will prompt you about creating it. (Multi-user Transana users will need to talk to their Database Manager about creating new databases.)

Change Document Splitter Orientation

Transana can display Documents side-by-side to allow comparison, and can display multiple simultaneous Transcripts for the same media files. By default, multiple Documents and Transcripts are shown in horizontal panes within the Document Window. If you would rather have these Documents and Transcripts displayed in vertical panes, go to the "Document" menu and select "Change Document Splitter Orientation".

Chat Window

The Multi-user version of Transana is designed to facilitate collaboration. The Chat Window, available in the "Tools" menu, is a simple text chat utility designed to allow researchers connected to the same database at the same time to communicate with each other via text chat. All connected users will be listed on the right side of screen. To chat privately, select names in that list; otherwise all users will see your messages.

Display Monitor

On dual-monitor computers, this setting in the "Options" menu allows you to select which monitor will display Transana.

Export Clip Video

When you create a Clip in Transana, what you create is a virtual Clip rather than an actual video file. However, it is occasionally useful to be able to export the media for a Clip as a separate video or audio file. To do this, right-click the Clip and choose "Export Clip Video". This should be done sparingly, as video files use up lots of hard drive space.

Please note that the Standard version of Transana does not support exporting Clip Video. To use this feature, you must be using the Professionsal, Computer Lab, or Multi-User version of Transana.

Fixed-Increment Time Codes

This function allows you to have Transana insert time codes at regularly spaced time intervals in a blank Transcript. Create a new Transcript and put that Transcript in Edit mode. Then go to the Document menu and select Fixed-Increment Time Codes. You will be prompted for the desired time increment. Enter the number of seconds you want between time codes.

You will notice that you will need to adjust your transcription style after using this function. You will need to use left-clicking to position in a time-coded segment of the Transcript followed by right-clicking (Ctrl-clicking on the Mac with a single-button mouse) the Transcript so that just the section of your media file that falls between consecutive time codes is played. Otherwise, it's difficult to keep the time codes positioned correctly relative to the Transcript text.

Insert Snapshot Hyperlink

This function allows you to insert a hyperlink to a Quote, Clip, Snapshot, or Note into a Document or Transcript. This is particularly useful when writing Research Journals.

Please note that Hyperlinks are not automatically removed from Documents and Transcripts when the linked Quotes, Clips, Snapshots, or Notes are deleted. Please also note that inserting a Snapshot Hyperlink is different from inserting a Snapshot into a Document or Transcript. Inserting a hyperlink give you only the link. Inserting a Snapshot gives you both the coded image and the hyperlink.

Load Snapshot Context

When a Snapshot is associated with a specific position in an Episode and Transcript, you can load that position in the media file by right-clicking the Snapshot in the Database Tree and selecting "Load Snapshot Context" in the popup menu.

Locate Quote in Document

There are times when it is desirable to be able to move from a Quote back to the Document it was created from. For example, you may want to look at what occurs immediately before or after a Quote, or you may want to create a similar Quote with different start or end points. To move from a Quote back to the original Document, right-click the Quote on the Database tab in the Data window and select "Locate Quote in Document."

Locate Clip in Episode

There are times when it is desirable to be able to move from a Clip back to the Episode it was created from. For example, you may want to look at what occurs immediately before or after a Clip, or you may want to create a similar Clip with different start or end points. To move from a Clip back to the original Episode, right-click the Clip on the Database tab in the Data window and select "Locate Clip in Episode." Please note that if the time codes associated with a clip are deleted in the originating Transcript, this function may not work properly.

Long Document Editing

Due to a bug in one of Transana's programming components, editing long Documents and Transcripts can become slow. The more of your Document or Transcript follows the current editing position, the slower the program becomes.

To work around this difficulty, go to the "Options" menu and select "Long Document Editing". This causes Transana to load only a segment of your active Document or Transcript when you go into Edit mode instead of the whole thing. This solves the problem. However, if you want to move to a different section of the document (moving more than about 200 paragraphs), you will want to leave Edit mode, move your cursor, and re-enter Edit mode.

Media Size

In the Options menu, the Media Size sub-menu allows you to change the size of the Media window and have all other Transana windows compensate automatically.

Record Lock Utility

The Multi-user version of Transana avoids problems with data conflicts between users by locking records when they are being edited. If Transana crashes while a user has one or more records locked, problems can arise. If you find that a record you need is locked by another user who is not currently logged into your database with Transana, you can go to the "Tools" menu and select the "Record Lock Utility". This utility assessed whether records can safely be unlocked and provides a mechanism to unlock those records that can be unlocked safely.

Play All Clips in a Collection

There are some circumstances when it is helpful to be able to review all Clips in a Collection in sequence. To do this, right-click the Collection and select the "Play All Clips" option. Clips will be played in the order they are displayed in the Database tab in the Data Window. To change the order in which Clips are played, simply cut-and-paste (or drag-and-drop) the Clips within the Collection so that they are displayed in the desired order. Play All Clips will play Clips from nested Collections automatically, but you can stop this by pressing the "Cancel" button.

Presentation Mode

Presentation Mode, selected in the "Options" Menu, offers four choices that affect the way Transana presents video during playback. "All Windows," the default setting, shows all Transana windows during video playback. "Video Only" displays the video window full screen when the video is playing. "Video and Transcript Only" displays the video in the top 65% of the screen and displays the Transcript, with scrolling highlight if Auto Word-tracking is enabled, in the bottom 35% of the screen. "Audio and Transcript Only" hides the video during playback, showing the Transcript full screen and playing only the audio portion of the media file.

Print a Transcript

The option allowing you to print the currently-loaded transcript is located in the Document Menu as well as the more traditional location of the File menu.

Quick Quote and Clip Mode

If "Quick Clip Mode" is checked in the "Options" menu, dragging a Keyword and dropping it on a Document or Transcript triggers the creation of a Quick Quote or Quick Clip. If "Quick Clip Mode" is not checked, dragging a Keyword and dropping it on the Document window applies that keyword to the currently-loaded Document, Episode, Quote, or Clip.

Show Quick Quote and Clip Warning

When Quick Quote or Quick Clip creation is triggered for a segment of a Document or Transcript that has already been used for a Quick Quote or Quick Clip, Transana will display a warning message about the duplication if this option is selected. Some researchers intentionally multi-code their Quick Quotes and Quick Clips and do not want such a warning slowing Quick Quote and Quick Clip creation down, while other researchers seek to code each Quote or Clip only once and find this warning very helpful.

Text Time Code Conversion

If you import a Transcript into Transana that was created in another transcription program, it might have time code information in HH:MM:SS.hh format. If you have not inserted any Transana time codes into the Transcript, you can use "Text Time Code Conversion", in the "Document" menu, to convert the text format time codes into Transana time codes. Please note that Transana time codes MUST be in linear order from the beginning to the end of the Transcript, and Transana does not support duplicate time codes. Any text time codes that do not conform to this will be left un-converted.

Update Database Window

You shouldn't even need this, but if your Data window becomes corrupted, you can right-click (Ctrl-Click on OS X) the root database node in the Database Tree and select "Update Database Window". This will re-load your Database Tree from the database.
