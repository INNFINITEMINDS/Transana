Converting Search Results to Collections

Transana lets you preserve the results of a search as a collection.

In this activity you convert the results of the Symmetry search, and convert those results into a collection called Symmetry Search Results.

Activity: Follow these directions:

1. Right-click the search name Symmetry. It's shown in the database tree with a magnifying glass icon next to it. Don't confuse it with the collection named Symmetry that is part of the result set for this search; this entry appears with a file cabinet icon next to it. Select Convert to Collection from the pop-up menu.

2. Transana opens the Collection Properties dialog box. Enter Symmetry Search Results in the field labeled Collection ID. Add your initials to the field labeled Owner and select Demonstration Keywords as the Default Keyword Group. Click OK when you have finished.

Notice that the Search Result disappears when the conversion is complete. Also note that Transana does not convert episodes in this process, because episodes cannot be included in collections even though they can be included in Search Results.
