Converting Search Results to Collections

Transana lets you preserve the results of a search as a collection. When you discover something brilliant using the Search in Transana, one way to save that analytic insight is through creating a collection full of the clips that support the new insight.

In this activity you convert the results of the Symmetry quick search into a collection called Symmetry Search Results.

Activity: Follow these directions:

1. Right-click the search name Tutorial : Symmetry. It's shown in the database tree with a magnifying glass icon next to it. Don't confuse it with the collection named Symmetry that is part of the result set for this search; this entry appears with a file cabinet icon next to it. Select Convert to Collection from the pop-up menu.

2. Transana opens the Collection Properties dialog box. Replace Tutorial : Symmetry with Symmetry Search Results in the field labeled Collection ID. Add your initials to the field labeled Owner and select Tutorial as the Default Keyword Group. Click OK when you have finished.

Notice that the Search Result disappears when the conversion is complete. Also note that Transana does not convert episodes in this process, because episodes cannot be included in collections even though they can be included in Search Results.

3. Finally, create a collection note to document the process by which you reached your brilliant analytic insight.
