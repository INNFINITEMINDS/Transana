Using Transana With PowerPoint

A number of people have asked if Transana Clips can be embedded in PowerPoint presentations the way edited video clips can be. The answer is that they cannot, primarily because Transana does not edit you raw video, but instead loads the full video and positions it according to the stored Clip data.

One option, if you are using Transana Professional, Multi-user Transana, or Transana for Computer Labs, is to use Transana's Export Clip Videofunction. This function will use Transana's Media Conversion tool to create a free-standing video file of the Clip's media contents. This file can be linked from PowerPoint, although you should be careful to keep the media file(s) with the presentation file if you will be presenting from a computer other than your own.

A second option is to run Transana and PowerPoint simultaneously, with Transana in Presentation Mode and all the Clips for the presentation copied to a custom Collection. You can use the Alt-Tab key combination on Windows or Command-Tab on the Mac to switch between your presentation and the display of your Clips within Transana. While it takes a little bit of practice to get used to this, the end result can be quite satisfactory.
