Merge Keywords

You may occasionally find it desirable to merge two keywords with each other. This might occur when you find yourself unable to distinguish between the Quotes, Clips, and Snapshots that each Keyword represents, or if you find that the two Keywords almost always co-occur and don't really seem to represent independent concepts.

To merge Keywords, right-click on the Keyword you want to merge into the other in the database tree, and choose "Keyword Properties." Edit the Keyword Group and Keyword so that they match an existing Keyword, the one you want to merge the selected Keyword into. Transana will ask you to confirm that you want to merge the two Keywords together, and, if you agree, will merge all instances of the first Keyword into the second, eliminating the first Keyword and all references to it in the process. The second Keyword's definition, Color, and other properties will be retained.

Please note that this is a permanent action that cannot be undone by Transana.
