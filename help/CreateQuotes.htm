Text - Creating Standard Quotes

If you are using the Standard version of Transana, you can skip this page. Working with text data is only supported in the Professional, Computer Lab, and Multi-User versions of Transana.

Once you've created a Collection, you can create a Standard Quote. Make sure you have the "Transana with Text" Document loaded and selected in the Document window to start with.

Activity: Follow these directions to create a Standard Quote.

1. Scroll down to the Standard Quotes section of the Document. The first paragraphs talk about creating a Collection, which we have already done. Look for a paragraph that states, "You can double-click the Collection you'd like to place your Standard Quote in to trigger Standard Quote creation" and select (highlight) that sentence by clicking the start of the sentence, holding your left mouse button down, dragging to the end of the sentence, and releasing the mouse button.

2. Leaving that selection in place in the Document window, move your mouse cursor to the Data window and double-click the Identifying What Is Important Collection.

3. You now see the Add Quote dialog box:

4. Enter "Create Quote by double-clicking Collection" in the Quote ID field.

5. Notice that your Quote is automatically assigned the Keyword Tutorial : Document, which is the Keyword that you assigned to the Document from which this Quote was created. We could add more Keywords to the Quote here if we wanted, but let's wait to do that.

6. Click OK.

7. Now right-click your new Create Quote by double-clicking Collection Quote and choose Add Quote Note from the popup menu. Enter "Create Quote by double-clicking Collection" as the Note ID and your initials under Note Taker. Press OK.

8. Press Ctrl-T to insert a date-time stamp into your Note, then enter analytic process information about the creation of the Quote. Close the Note when you are finished.

Note that your analytic memos are a place where you can record ideas for future investigation as they come up, but if you do this, you will want to flag them with an identical tag, e.g., ("FOLLOW UP"). This makes it much easier to find them in the future using the Notes Browser's Text Search capability.

Lastly, if you want to make a Quote from a single sentence, you don't need to select the text. Instead, place the Document cursor somewhere in the sentence and create your Quote. Transana will automatically use the sentence for the Quote in the absence of a manual selection.

Activity: Follow these directions to create another Standard Quote in a different way.

1. Look for the sentence that states 'Right-click the Collection and choose "Add Quote" from the menu' and place the document cursor in it by clicking somewhere in the sentence. You do not need to make a selection, as we just want one sentence.

2. Right-click the Identifying What Is Important Collection and select Add Quote from the popup menu.

3. You now see the Add Quote dialog box, similar to the one shown above (but with different Quote Text.

4. Enter "Create Quote by right-clicking Collection" in the Quote ID field.

5. Notice that your Quote is automatically assigned the Keyword Tutorial : Document, which is the Keyword that you assigned to the Document from which this Quote was created. We would like to add a Keyword that identifies this as a Quote, but no such Keyword exists. Fortunately, we can add new Keywords while in the midst of creating a Quote.

6. To do this, press the Keyword Management button, which looks like this , on the Add Quote form. This will open the Keyword Management tool.

7. Press the Add Keyword to List button and add a Tutorial : Quote Keyword. Press OK to close the Add Keyword dialog and Close to close the Keyword Management form. You now have a Quote Keyword in the Tutorial Keyword Group. Double-click it to add it to the Quote Keywords list.

8. Click OK.

9. Now right-click your new Create Quote by right-clicking Collection Quote and choose Add Quote Note from the popup menu. Enter "Create Quote by right-clicking Collection Note" as the Note ID and your initials under Note Taker.

10. Press Ctrl-T to insert a date-time stamp into your Note, then enter analytic process information about the creation of the Quote. Close the Note when you are finished.

We need several Quotes for demonstration purposes later in the Tutorial, so let's create one more.

Activity: Follow these directions to create another standard Quote in a different way.

1. Look for the sentence that states, "Drag the selected text from the Document and drop it on the target Collection" and select that sentence.

2. Now click on the Document selection, holding down your left mouse button, and drag the selection onto the Identifying What Is Important Collection in the Data window.

3. Once again, you see the Add Quote dialog box. Enter "Create Quote by dragging selection to Collection" in the Quote ID field.

4. Add a Tutorial : Quote Keyword. Click OK.

5. Add an appropriate Quote Note to document your analytic process.

You now have a Collection containing three Quotes. They were all taken from the same Document, but were created in slightly different ways.
