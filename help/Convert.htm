Converting a Search Result to a Collection

There are two ways to indicate the analytic significance of Quotes, Clips, and Snapshots in Transana. You can place these items in a Collection to indicate their analytic significance, or you can apply Keywords to them to code them. Collections seem more tangible to some researchers, while Keywords seem more flexible, in that you can describe a data item along multiple dimensions using multiple keywords, and keywords are accessible to the Reports, Maps and Graphs, and Search.

To allow these two different systems to work together, Transana provides a way to convert a Search Result, which is based on Keywords, into a Collection. Right-clicking a Search Result gives you the option to convert the Search Result into a Collection. Please note that while Documents and Episodes can be included in Search Results, they are dropped when those results are converted to a Collection.

A second way to preserve a discovery that occurs while searching is to Save the Search in the Search interface.
