Media - Add a Transcript

In Transana, the term Transcript refers to any written record that somehow corresponds to a media file in a linear way. There are many different ways that Transcripts can be used analytically.

Whenever you add an Episode, Transana automatically prompts you to create a Transcript. The dialog box looks like this:

Activity: To create a transcript, enter Transcript in the Transcript ID field, and your name or initials as the Transcriber. Click OK.

A dialog asking about Transana Wave File creation will open. Say Yes so that Transana can extract the audio from your media file and create the appropriate waveform diagram.

If you have an existing transcript for your video, you can save that transcript as a Word Document (*.docx) or a Rich Text Format (*.rtf) file in most word processors, then import it into Transana through this screen. If the document is in a language that can be encoded with Latin-1 encoding, you can also save documents in plain text (*.txt) format for import into Transana.

Advanced Topic: Some researchers use multiple simultaneous transcripts to implement multiple analytic layers. See Working with Multiple Transcripts in the Transana Manual for more information. Multiple simultaneous transcripts are supported in the Professional, Computer Lab, and Multiuser versions of Transana.
