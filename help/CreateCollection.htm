Creating a Collection

Clips are organized into collections. A collection is simply a container used for holding clips that are somehow related to each other. Therefore, before you can identify a clip, you need to create a collection to put it in.

Activity: Follow these directions to create a collection.

1. In the Data window, right-click the Collections node, and select Add Collection. You see a dialog box that looks like this:

2. Type Symmetry in the field labeled Collection ID.

3. Enter your name or initials in the Owner field, and select the Demonstration Keywords from the Default Keyword Group list. Click OK when you are done.
