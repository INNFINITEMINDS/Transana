Creating New Collections from Existing Clips

You can create collections manually from existing clips. You can move clips from one collection to another by selecting a clip and using cut-and-paste (or drag-and-drop on Windows) to move it to the new location. (On Windows, hold down the Ctrl key when you drop the clip to copy it instead of moving it; on Windows or Mac, use Copy to copy instead of Cut to move the clip.)

Activity: Follow these directions to create a new collection and populate it with existing clips.

1. In the Data window, right-click the Collections node, and select Add Collection from the pop-up menu.

2. Type Kids' Ideas About Symmetry in the field labeled Collection ID.

3. Enter your name or initials in the Owner field, and select the Demonstration Keywords from the Default Keyword Group list. Click OK when you are done.

4. Click on the clip called Houses from the Symmetry collection.

Windows-only Alternative: Hold down the Ctrl key, then drag it to the new collection called Kids' Ideas About Symmetry. Be sure to continue holding down the Ctrl key when you release the mouse button.

Macintosh or Windows Alternative: Right-click Houses and select "Copy" from the popup menu. Then right-click Kids' Ideas About Symmetry and select "Paste" from the popup menu.

Transana opens a dialog box that asks you if you want to copy the clip. Click Yes. If it asks if you want to move the clip, say No and try again.

5. Use this same procedure to copy the clips called The Same, and Folding to the Kids' Ideas About Symmetry collection.

Note that you can also move clips instead of copying them. To move a clip, use "Cut" instead of "Copy", or on Windows only, don't hold the Ctrl key when you drop the clip.

You can copy or move all the clips in a collection, by using cut/copy and paste (or on Windows using Drag and Drop) with a collection name within the database tree.
