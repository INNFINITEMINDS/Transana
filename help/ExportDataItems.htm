Analytic Data Export

Transana's Analytic Data Export routine creates a tab-delimited text file that allows you to look at information about Quotes and Clips you've created in Transana with other programs such as SPSS and Excel.

If you call the Analytic Data Export routine from a Library, Transana will create a listing of all Quotes and Clips created from all Documents and Episodes in the Library regardless of what Collections those items are placed in, and sends it to the Filter Dialog, along with a list of all Keywords applied to that set of items.

If you call the Analytic Data Export routine from a Document, Transana creates a listing of all Quotes created from that Document, regardless of what Collections those Quotes are placed in, and sends it to the Filter Dialog, along with a list of all Keywords applied to that set of Quotes.

If you call the Analytic Data Export routine from an Episode, Transana creates a listing of all Clips created from that Episode, regardless of what Collections those Clips are placed in, and sends it to the Filter Dialog, along with a list of all Keywords applied to that set of Clips.

If you call the Analytic Data Export routine from a Collection, Transana sends a list of all Quotes and Clips gathered together in that Collection, regardless of what Documents and Episodes they are from, and sends it to the Filter Dialog, along with a list of all Keywords applied to that set of items.

The first line of this text file is made up of column headings or variable names. Each subsequent line represents an individual Quote or Clip. Note that Snapshots are not included in Analytic Data Export, as the data required to describe them is too different to be integrated into Analytic Data Export in a way that would be useful.

The first column of this text file is the name of the Collection where the Quote or Clip is stored. Nesting information for that Collection is provided. The second column is the type of item in the row, 1 indicating Quote and 2 indicating Clip. The third column is Item Name, and the fourth column is the name of the text file that served as the source of the Quote or the media file associated with the Clip. The fifth and sixth columns are the Start and Stop position, in characters for Quotes and in H:MM:SS.s format for Clips. The seventh column is the length in characters for Quotes or seconds, accurate to the frame and listed in thousandths of a second, for Clips. Note that a Quote length CAN be 0 if the source Document was edited to remove the text the Quote was taken from. Subsequent columns are the coding data for the items, with the Keyword Group and Keyword indicated in the first row of the column, and with a 0 indicating that the specified keyword was not applied to the specified item and a 1 indicating that the keyword was applied to the item.

Please note that because analytic data is exported to a tab-delimited text file, there can be issues with character encoding for characters that are not included in the English alphabet. Transana exports data to this file using UTF-8 encoding.
