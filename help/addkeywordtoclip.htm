Adding a Keyword to an Existing Clip

It is quite likely that you will want to add new keywords to existing clips at some point in your analysis. For example, we created a new keyword while creating the clip Houses that we probably want to apply to the Introduction clip. Because this sort of coding is such a central analytic activity, Transana provides several ways to do this.

First, create two new keywords in the Tutorial keywords group. Add the Keywords Introduction and Definition to the Tutorial group. (See Creating Keywords if you cannot remember how to create new keywords.)

Activity: Follow these directions to add a keyword to a clip. The quickest method is to use the database tree in the Data window to apply keywords to clips.

1. Make sure the Keywords node in the database tree is fully expanded. (If necessary, click the + or triangle next to Keywords, then the + or triangle next to Tutorial.)

2. Click to select the keyword Introduction from the list of keywords.

Drag-and-drop option: Drag the keyword Introduction up and drop it on the clip Introduction under the Symmetry collection in the database tree.

Right-click option: Right-click the keyword Introduction and select "Copy" from the popup menu. Then right-click the Clip Introduction in the Symmetry collection in the database tree. Select the "Paste" option from the resulting menu.

Transana displays a dialog box asking you to confirm that you want to add the keyword. Click Yes.

Note that you can add a keyword to all the clips in a collection by pasting or dropping the keyword on the collection name. Drag or copy the keyword Symmetry onto the collection Symmetry to ensure that all of the clips in that collection include that keyword. Note that clips that already have the Symmetry keyword will not be changed because of this duplication, but also note that clips you add to this collection later are not automatically given the Symmetry keyword.

You can also add keywords to episodes and to all episodes in a series using copy-and-paste or drag-and-drop within the Database tab of the Data Window.

Sometimes you might want to view a clip and its transcript while deciding what keywords are appropriate. Use this method to add the keyword Definition to the clip called Introduction.

Activity: Follow these directions:

1. Load the clip by double-clicking the clip Introduction in the collection Symmetry in the database tree.

2. First, check in the Options menu to make sure that Quick Clip Mode is not checked. Drag the keyword Definition from the Tutorial keyword group under the Keywords root node in the database tree, and drop it on the Transcript window. Transana opens a dialog box that asks you to confirm that you want to add the keyword. Click Yes. (If Quick Clips Mode is checked, this action will create a Quick Clip rather than applying the keyword to the Episode or Clip currently loaded.)

Also notice the Keywords tab in the Data window. This tab displays a list of keywords associated with the clip or the episode that is loaded in the Transana main interface windows. This tab allows you to edit or delete keywords from a clip or an episode.

3. Click the Keywords tab in the Data window. Transana shows the keywords associated with the Introduction clip.

4. Right-click anywhere on this tab and choose Edit from the pop-up menu.

Transana opens the Edit Keyword List dialog box, which looks like this:

5. If the keyword group : keyword pair Tutorial : Symmetry is not already listed in the Keywords box, select the Tutorial keyword group, then double-click the Symmetry keyword to add it to the clip. Click OK if you added the keyword, Cancel if you did not.

Transana also allows you to add or delete keywords for episodes and clips from the Properties dialog box that is associated with each episode or clip.

Activity: Use the database tree in the Data window. Follow these directions to add a keyword to the Demo episode:

1. Right-click the Demo episode in the Geometry by Design series under the Series root node in the database tree, and choose Episode Properties from the pop-up menu.

2. In the dialog box that appears, click on Tutorial keyword group in the field labeled Keyword Group. Then double-click Symmetry in the field labeled Keyword to add it to the Episode. You should see the keyword group : keyword pair appear in the Episode Keywords box to signal that the keyword has been added to the episode. Click OK when you are done.

If this were a real analysis, you would want to edit your analytic memos as you were going along making these coding changes. In each appropriate note, you would add a date/time stamp and an explanation of the coding changes performed for each data object within Transana.

Activity: Because this is a practice analysis, let's just update one note.

1. Locate the analytic memo note that you created earlier for the Introduction clip. Double-click this note to open it for editing.

2. Decide if you want to add new commentary to the top of the note or to the bottom and place your cursor at the appropriate location. Press Ctrl-T to add a date/time stamp, and document your decision to add the Symmetry, Introduction, and Definition keywords to this clip. It is a good idea to leave a blank line between process notes from different days in your analytic memos. Exit the note when you are done making your changes.
