Transcript Properties

To add a Transcript in Transana, right-click (Ctrl-click on the Mac) an "Episode" node in the Data window and choose "Add Transcript." You will see the dialog box shown above. The only required information is a "Transcript ID" that is unique within the Episode. "Transcriber" can be used to indicate the name of the person who created the Transcript, and "Title/Comment" can be used in any way you wish. You can import Rich-Text Format (*.rtf) documents using the "File to Import" field and the associated "Browse" button.
