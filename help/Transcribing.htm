Transcribing a Media File

Transcription within Transana is a manual process. Voice recognition technology is not sophisticated enough in most instances to produce anything useful from the data researchers collect in most research settings. With a little practice you will find that Transana's transcription tools make the process faster and easier than many transcription options.

The teacher in the Demo.mpg video is discussing the manipulation of the core squares of a quilt with the students. Their current activity is to observe what happens when she flips the core square, and later, she and the students will discuss symmetry in two-by-two designs.

Activity: Follow these directions:

1. In Transcription window's toolbar, click the Edit Mode icon, which looks like this: . When you do this, the Transcription window is put into edit mode and all the icons on the Transcription toolbar are enabled.

2. Press Ctrl-S to start the video. Listen for a few seconds, then press Ctrl-S again to pause the video. Type what you heard.

3. Press Ctrl-S again to restart the video. Transana automatically rewinds the video two seconds to give you time to get your bearings. This is a standard feature of transcription devices.

Continue to use Ctrl-S to start and stop the video as necessary as you transcribe the audio. Insert a blank line in the transcript each time the speaker changes. You will find it much easier to work with your transcript if you add this visual demarcation of speaker change.

Note these other features you might find useful:

Ctrl-A starts media file playback with a ten-second rewind.

Ctrl-D pauses or starts media file playback with no rewind.

Ctrl-F moves the media file forward by ten seconds and plays it.

Ctrl-(comma) slows down media file playback by 10% of normal speed, (if playback speed change is possible.)

Ctrl-(period) speeds up media file playback by 10% of normal speed, (if playback speed change is possible.)

4. Click the Save icon periodically. Be sure to save when you have finished transcribing.
