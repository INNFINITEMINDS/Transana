Add an Episode

Once you've created a series, you can add an episode. Use the sample video file that you downloaded from the Transana Web site.

Activity: Follow these directions:

1. To add an episode, right-click on Sample under the Series node, then choose Add Episode from the menu.

You see the following dialog box.

Notice that the Series ID is already filled in, and that your Default Keyword Group and Keywords are shown.

2. In the Episode ID field, enter Sample Video and click the Browse button to select the Demo.mpg video file in the C:\Program Files\Transana 2\Sample folder on Windows or the HOME/Movies/Sample folder on the Mac (or wherever you saved it.)

3. Double-click on the Geometry and Quilt keywords to assign these keywords to this Episode. (This video is a lesson on teaching geometry through a quilting exercise, and adding those keywords to the episode will cause them to be used as the default keywords for all clips made from this episode.)

4. Click OK. Transana displays the Add Transcript dialog box.
