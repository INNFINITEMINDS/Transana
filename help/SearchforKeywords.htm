Searching for Keywords

Transana lets you search for clips and episodes that contain specified keywords. This is useful when you want to look for evidence to support a certain hypothesis, especially when you have large amounts of video data.

Searches have names, and, once created, they appear in reverse chronological order under the Search node in the database tree. This lets you keep track of what you've searched for, and what you found when you searched. You can save the results of a search as a collection.

Activity: Follow these directions to perform a search:

1. Right-click the Search node in the database tree in the Data window and choose Search from the pop-up menu. Transana opens the Boolean Keyword Search dialog box

2. Type Symmetry in the field labeled Search Name.

3. Click Symmetry in the list of keywords in the dialog box. You might have to scroll down to see this keyword.

4. Click Add Keywords to Query.

5. Click Search to begin searching.

Transana finds one episode and six clips that have the keyword Symmetry assigned to them. It shows you the result of the search under Symmetry, under the Search node. Notice that the search name appears under the Search node with an icon of a magnifying glass next to it.

Notice that the series/episode and collection/clip structure you have set up is preserved in the search results. This search returns a small set of results. The next activity uses Boolean operators.

Activity: Perform a second search using the keywords Symmetry and Confusion. Follow these directions:

1. Right-click the Search node in the database tree in the Data window and choose Search from the pop-up menu. Transana opens the Boolean Keyword Search dialog box

2. Type Symmetry and Confusion in the field labeled Search Name.

3. Click Symmetry in the list of keywords in the dialog box.

4. Click Add Keywords to Query.

5. Click the AND button under the label Operators in the dialog box.

6. Click Confusion in the list of keywords in the dialog box. Add it to your Search.

7. Click Search to begin searching.

Transana displays the results of your search in the database tree.

This search produced four clips, which you can review by right-clicking the Symmetry collection under the Symmetry and Confusion Search Result node and selecting Play All Clips. In one clip, a student talks about the carpet being symmetrical if it's all gray. In the second, a student suggests that symmetry may have something to do with houses. In the third, the class discusses symmetry as meaning things that are the same, and in the fourth, the teacher is asking about how students will know if they can always create symmetrical two-by-two designs.

The clip Proof in this instance is a false positive result. In this clip, the students are clear about the concept of symmetry, but are expressing confusion about the broader implication of their initial finding that they can always create symmetrical two-by-two designs. Let's drop it from your Search Result (but not from its Collection) by right-clicking the clip in the search result and selecting Drop for Search Result from the pop-up menu.
