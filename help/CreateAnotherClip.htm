Create Another Clip

In this part of the tutorial, you add another clip, and create a new keyword. If your Sample Transcript for the Sample Video is not currently loaded, you should load it now. Remember, you load a transcript and its associated episode by double-clicking on the transcript item in the database tree in the Data window.

Make sure that the Transcription window is not in Edit mode. If necessary, turn off Edit mode by clicking the Edit button on the Transcription window toolbar. You also need to have time codes visible in the transcript, so if the time codes are hidden, click the time code button on the toolbar to display them.

In this part of the tutorial you create a clip that lacks the appropriate time codes. Creating this kind of clip is a two-step process. First, you must insert time codes before and after the text you want to make a clip. Then you have to create the clip itself.

Starting at 3:34.7, the teacher begins a discussion of symmetry with the class. Your next clip should begin with the statement "Let's talk about that word, symmetrical" and run through the question "What about houses would symmetrical have to do with?" (ending at 3:53.0). Begin by inserting some time codes on either side of this exchange.

Activity: Follow these directions to insert some new time codes:

1. Move the crosshair cursor in the visualization window until you are at 3:34.7 and click as close to that time as possible in the waveform. Remember that Transana shows the time in the field labeled Time: in the status bar of the Visualization window. (Don't worry if you can only click at 3:34.5 or 3:34.9; just try to get as close as possible. If precision were required, we could always zoom in on the Visualization Window to make the precise selection easier.) Notice that Transana positions the video at 3:34.7 also.

2. Notice that Transana has highlighted the portion of the transcript where the teacher is reading aloud to the students. This is because this section is surrounded by time codes, which you inserted at the beginning of each change of speaker. (If Transana did not move to the appropriate point in the transcript, you were probably in Edit mode. Transana does not move the Transcript cursor based on a Waveform selection in Edit Mode, as that makes transcription more difficult.)

3. Once you have the video positioned, the next thing you need to do is put the Transcript in Edit mode so that you can insert the time code. Press the Edit Mode button .

It is important to note that when Transana is not in Edit Mode, clicking in the waveform changes the video position and moves the highlight in the transcript. Similarly, clicking in the transcript changes the video position and moves the highlight in the waveform. Clicking in the Waveform or the Transcript, when not in Edit Mode, alters the position of the video. This is useful during analysis of the video, where you want Transana's windows to remain synchronized at all times.

However, when you are in Edit mode, Transana does not alter the position of the video or the position of the highlight in other windows based on clicks. This is because when you are trying to edit your transcript and insert time codes, you're trying to get the video position and the cursor in the Transcript to match, and if clicking either window altered the other, this would be impossible. This distinction is admittedly rather subtle, and is something that new Transana users sometimes struggle with until they get used to it.

4. You need to unselect the highlighted portion of the transcription, because you are about to enter a new time code, so click anywhere in the unhighlighted portion of the transcription.

5. Move the cursor in the Transcription window to just before the statement "Let's talk about that word, symmetrical." And click to position the cursor.

6. If you want to confirm the position of the video, press Ctrl-S. This will rewind the video by 2 seconds and start playing it. You can then press Ctrl-S again when the teacher is finished saying that Amanda asked "What's symmetrical?" Then press Ctrl-T to insert a time code before the word "let's."

7. Now press Ctrl-D and let the video play for about 20 seconds, stopping playback with Ctrl-S or Ctrl-D just after the teacher finishes asking what about houses symmetrical would have to do with at 3:53.0.

8. Move the cursor in the Transcription window to just after the word "with?" in the sentence "What about houses would symmetrical have to do with?" Click to position the cursor, then press Ctrl T to insert another time code, if there's not already one in that position.

Activity: Follow these directions to create the clip:

1. Highlight the transcription text between these two time codes that you just inserted.

2. Windows-only Alternative: Click and drag the highlighted text from the Transcript Window to the collection called Symmetry in the database tree in the Data window.

Macintosh or Windows Alternative: Right-click the Collection where you want to create the Clip (Symmetry) and select "Add Clip".

Transana opens the Add Clip dialog box.

3. Enter Houses in the field labeled Clip ID.

4. Look in the field called Clip Keywords for the keyword Symmetry. If it's not already there, apply the keyword "Symmetry" to this clip by double-clicking on Symmetry in the Keyword list.

5. It is clear from the video that the student named Paul does not understand the concept of symmetry at this point, and you should indicate that in your keywords. You don't yet have a keyword that describes that, so you should add one now. Click the key icon in the Add Clip dialog box. Transana opens the Keyword Management dialog box.

6. Click Add Keyword to List. Transana opens the Add Keyword dialog box.

7. In the field labeled Keyword, enter Confusion. If you want, enter a short definition in the field labeled Definition. Click OK when you are done.

8. Click Close to leave the Keyword Management dialog box.

9. Double-click Confusion in the Keyword field of the Add Clip dialog box. Then click OK to leave this dialog box.
