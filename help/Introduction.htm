Introduction and Overview

Transana is designed to facilitate the qualitative analysis of data, including text, video, audio, and still image data.

Transana allows researchers to:

  • bring text, media (video and audio), and still image data into the analytic environment

  • create analytic transcripts of media files, and link the transcripts to the media file contents

  • identifying, manipulate, organizing, and code analytically interesting portions of text, media files, and image files

  • create a variety of reports and graphs for exploring and interpreting coded text, media, and image data

  • search coded data by keyword and by combinations of keywords

  • organize large data sets easily

  • collaborate with other researchers, near and distant, by allowing simultaneous access to analytic data by multiple researchers with real-time data updating (Multi-user version only)

Transana pays special attention to the analytic needs of researchers with media data, offering several unique, powerful tools for working with audio and video files. It allows researcher with media data to:

  • connect multiple overlapping media files together for simultaneous playback and analysis

  • create multiple transcripts for media files which can be displayed simultaneously to facilitate the exploration of media files using multi-layered, multi-modal approaches

These activities allow researchers to do qualitative analysis using a wide variety of methodologies and theoretical approaches.

Transana is developed and maintained by David K. Woods, Ph.D. at the Wisconsin Center for Education Research, University of Wisconsin-Madison.
