Introduction and Overview

Transana is designed to facilitate the qualitative analysis of video, audio, and still image data. Transana allows researchers to:

  • view video, listen to audio, and view images

  • navigate through large data sets easily

  • create analytic transcripts of media files, and link the transcript to file contents

  • identifying and organizing analytically interesting portions of media and image files

  • code video, audio, and still images analytically

  • view a variety of reports and graphs for exploring and interpreting coded media and image data

  • search video, audio, and image by keyword and by combinations of keywords

  • create multiple transcripts which can be displayed simultaneously to facilitate the exploration of media files using multi-layered, multi-modal approaches

  • connect multiple media files together for simultaneous playback and analysis

  • collaborate with other researchers, near and distant, by allowing simultaneous access of analytic data by multiple researchers with real-time data updating (Multi-user version only)

  • manage the organization and storage of large collections of digitized video, audio, and image files

These activities allow researchers to do qualitative research using a wide variety of methodologies and theoretical approaches.

Transana was originally created by Chris Fassnacht. It is now developed and maintained by David K. Woods at the Wisconsin Center for Education Research, University of Wisconsin-Madison.
