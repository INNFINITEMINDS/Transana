Introduction and Overview

Transana is designed to facilitate the transcription and qualitative analysis of video and audio data. It provides a way to view video, create a transcript, and link places in the transcript to frames in the video. It provides tools for identifying and organizing analytically interesting portions of videos, as well as for coding those video clips. It provides a variety of reports and graphs for examining coded video, as well as a mechanism for searching for portions of analytically interesting video by keyword and by combinations of keywords. It also features database and file manipulation tools that facilitate the organization and storage of large collections of digitized video.

More specifically, it allows users to:

  • Import and view video in MPEG-1, MPEG-2, AVI, MOV, and MP4 formats on Windows and Mac OS X, and WMV format on Windows only. Transana also works with MP3 and Wave audio on Windows and OS X and WMA audio on Windows only.

  • Easily create transcripts of the media files, either audio or video. Special symbols for Jeffersonian Transcription are readily accessible, but are not required.

  • Navigate through media files using several different mechanisms including a precise Waveform image.

  • Link points in a media file to positions in a transcript by embedding time codes in the transcript.

  • Automatically highlight the relevant portion of the transcript while the media file plays.

  • Locate the portion of a media file that corresponds with a selected section of the transcript.

  • Select analytically interesting portions of media files, which can be organized into meaningful groups.

  • Define keywords and apply them to portions of media files.

  • Create text reports about clips and coding, as well as graphs of coding across one or more media files.

  • Search for instances of keywords or combinations of keywords and see or hear the media clips to which they have been applied.

  • Generate a variety of reports about the analysis performed on media files.

  • Create multiple transcripts which can be displayed simultaneously to facilitate the exploration of media files using multi-layered, multi-modal approaches.

  • Connect multiple media files together for simultaneous playback and analysis.

  • Collaborate with other researchers, near and distant, by allowing simultaneous access of analytic data by multiple researchers with real-time data updating. (Multi-user version only.)

  • Securely transfer media files to and from a Storage Resource Broker (SRB), a computer system with massive storage capacity, for collaboration purposes.

These activities allow researchers to do qualitative research using a wide variety of methodologies and theoretical approaches.

Transana was originally created by Chris Fassnacht. It is now developed and maintained by David K. Woods at the Wisconsin Center for Education Research, University of Wisconsin-Madison.
