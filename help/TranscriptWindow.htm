Transcript Window

The transcript window is where you can view or edit your transcripts. You can add time codes, which link a frame in the video to a spot in your transcript. Once this is done, the highlight in the transcript will move as the video is played, highlighting the appropriate section of the transcript, if the transcript is not in "Edit Mode."

The transcript window includes speed buttons for inserting symbols commonly used in Jeffersonian Transcript Annotation. These symbols are , which indicates rising intonation, , which indicates falling intonation, , which signifies an audible intake of breath, and , which signifies a whisper or quieted speech. (See Transcript Notation in Transana's Help Menu for more information about Jeffersonian Transcription Symbols.)

The next set of buttons are the Show/Hide Time Codes toggle, denoted by the time code icon (¤) and the Edit Mode/Read-Only toggle, denoted by the icon. Please note that the edit button must be pressed, placing the document in Edit Mode, before you can transcribe anything or insert time codes.

The final set of buttons in the Transcript Window speedbar are the Keyword button , which allows you to edit the Keyword List associated with the currently-loaded Episode or Clip, the Save button , which saves the current transcript, and the Select Clip Text button which indicates that text highlighted in the Transcript should be selected for creating a Clip.

If you have time codes in your transcript, you can right-click the transcript to play the section between the nearest time codes where the cursor is currently placed. You can also highlight a larger section of the transcript if you want to play a portion that crosses time codes. In addition, if you would like to play from the current position in the transcript to the end of the loaded Episode or Clip, you can hold down the Ctrl key and right-click the Transcript on Windows or the Ctrl and "Command" or "Meta" key and click the Transcript on the Mac.

During transcription, you may find it easiest to use Ctrl-S to start and stop video playback. This causes the video to be played in Transcription Mode, which causes an auto-rewind to be performed when play is initiated. You can set the amount of the rewind, as well as the playback speed, using the Transcriber Settings tab on the Program Settings dialog, which is located in the Options menu. In addition, Ctrl-D will pause and restart the video playback without auto-rewinding, Ctrl-A will rewind the video by 10 seconds, Ctrl-F will fast-forward the video by 10 seconds, Ctrl-N will cause the next time-coded segment to be played, and Ctrl-P will cause the previous time-coded segment to be played. Finally, Ctrl-T will cause a time code for the current video position (as indicated in the Sound Visualization Window) to be inserted at the current cursor position in the Transcript.

Here is a summary of the keyboard shortcuts available in the Transcription Window:

Ctrl-XCut highlighted text
Ctrl-CCopy highlighted text
Ctrl-VPaste text
Ctrl-ARewind 10 seconds and play video.
Ctrl-SIf video is playing, pause it. If not, rewind video by the amount of the Transcription Setback setting in Program Settings (2 seconds by default) and play it.
Ctrl-DIf video is playing, pause it. If not, play it with no rewind.
Ctrl-FFast-Forward the video 10 seconds and play it.
Ctrl-PPlay the previous time-coded segment.
Ctrl-NPlay the next time-coded segment.
Ctrl-BToggle Bold character formatting.
Ctrl-IToggle Italics character formatting.
Ctrl-UToggle Underline character formatting.
Ctrl-Cursor UpInsert Rising Intonation symbol.
Ctrl-Cursor DownInsert Falling Intonation symbol.
Ctrl-HInsert Audible Breath symbol.
Ctrl-OInsert Whispered Speech symbol.
F12Quick Save (unless intercepted by Dashboard on OS X 10.4.)
