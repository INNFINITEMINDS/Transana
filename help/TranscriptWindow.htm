Transcript Window

The transcript window is where you can view or edit your transcripts. You can add time codes, which link a frame in the video to a spot in your transcript. Once this is done, the highlight in the transcript will move as the video is played, highlighting the appropriate section of the transcript (if the transcript is not in "Edit Mode.")

The transcript window includes speed buttons for inserting symbols commonly used in Jeffersonian Transcript Annotation. These symbols are , which indicates rising intonation, , which indicates falling intonation, , which signifies an audible intake of breath, and , which signifies a whisper or quieted speech. (See Transcript Notation in Transana's Help Menu for more information about Jeffersonian Transcription Symbols.)

The next set of buttons are the Show/Hide Time Codes toggle, denoted by the time code icon , and the Show/Hide Time Code Values, denoted by the  icon. The first of these buttons controls whether the time code symbol is displayed in the transcript, while the second one controls whether the time code value, in human-readable form, is displayed. (Please note that it is not a good idea to edit the time code value text. This will not affect the underlying time code, but may cause display problems when Transana tries to hide this automatically-generated text.)

The Edit Mode/Read-Only toggle, denoted by the  icon, is next. Please note that Transana starts up in "read-only" mode, where the highlight selection in the transcript moves with the media file as it plays. Pressing this button toggles you into the "edit mode," where you can transcribe and insert time codes. Having the transcript highlight track the media file position is very useful when you are watching or listening to your media file, but when you are editing a transcript, you don't want the cursor moving around in the transcript when you play video, as this is very disruptive to editing.

The next set of buttons in the Transcript Window toolbar are: the Create Quick Clip button , which allows you to create Quick Clips if you have a Keyword selected in the Database Tree; the Keyword button , which allows you to edit the Keyword List associated with the currently-loaded Episode or Clip; and the Save button , which saves the current transcript.

The Propagate Changes button is next, as indicated by , which allows you to push changes in an episode transcript out to clips covering that portion of the episode or to push changes in a clip transcript out to copies of the currently-loaded clip.

The Match Selection in Other Transcripts button, shown by the  icon, causes all transcript selections to match the one in the selected transcript window when multiple transcripts are open at the same time. The Play button, shown by the  icon, causes the current selection(s) to be played, which can be particularly useful when creating multiple-transcript clips.

If you have time codes in your transcript, you can right-click the transcript to play the section between the nearest time codes where the cursor is currently placed. You can also highlight a larger section of the transcript if you want to play a portion that crosses time codes. In addition, if you would like to play from the current position in the transcript to the end of the loaded Episode or Clip, you can hold down the Ctrl key and right-click the Transcript on Windows or the Ctrl and "Command" or "Meta" key and click the Transcript on the Mac.

During transcription, you may find it easiest to use Ctrl-S to start and stop video playback. This causes the video to be played in Transcription Mode, which causes an auto-rewind to be performed when play is initiated. (You can set the amount of the rewind, as well as the playback speed, using the Transcriber Settings tab on the Program Settings dialog, which is located in the Options menu.) In addition, Ctrl-D will pause and restart the video playback without auto-rewinding, Ctrl-A will rewind the video by 10 seconds, Ctrl-F will fast-forward the video by 10 seconds, Ctrl-N will cause the next time-coded segment to be played, and Ctrl-P will cause the previous time-coded segment to be played. Finally, Ctrl-T will cause a time code for the current video position (as indicated in the Visualization Window) to be inserted at the current cursor position in the Transcript.

Advanced Topic: Some researchers like to use tabs while transcribing in an effort to line up columns of text. If you would like to use horizontal alignment, here are some helpful hints for this style of transcribing in Transana.

Before you create your transcript, use the Transcriber Settings tab of the Program Options dialog to set Transana's Default Font to a fixed width font. We recommend Courier New font. You will want to set your Tab Size to be larger than your speaker identifiers. "T:" works with tab size 4, while "Teacher:" requires a tab size of at least 10. Finally, be sure to turn Word Wrap off, as you need to set line lengths manually when transcribing this way.

Once you create your transcript, you need to use only fixed-width fonts, and it is probably easiest if you use only a single size of a single font. You will also need to limit the length of your lines. We have found that an upper limit of 80 characters per line if you are using Courier New 10 point font and 64 - 66 characters per line if you are using Courier New 12 point font seems to work well. You may need to do some experimenting to determine what works best for you.
