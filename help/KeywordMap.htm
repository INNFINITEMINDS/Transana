Keyword Map

The Keyword Map presents a visual display of where keywords have been assigned to Clips created from the selected Episode. Each colored band in the Keyword Map shows the start and stop points of a Clip, along the horizontal timeline of the Episode, where a particular keyword, along the vertical axis, has been applied. Bands of the same color with the same starting and ending points indicate that a single clip was assigned multiple keywords.

The Keyword Map has several options in its toolbar (and the menu on Windows only.) You can save a copy of the Keyword Map diagram in JPEG format. The JPEG image created will have the same width and at least the same height as the Keyword Map window on the screen, so you may want to resize the window prior to saving your image. Similarly, the dimensions of the window on screen affect the overall shape of the printout when Print Preview or Print is selected from the toolbar or menu. Some experimentation may be required to get the exact image size and proportions you desire in a saved graphic or a printout.

At this time, this report has one major implementation flaw. The Keyword/Timeline bands are drawn opaquely, overwriting previous band information, so if a particular keyword was applied to more than one clip with overlapping time segments, only the last clip will be visually apparent. It is possible that multiple clips share the same keyword / timeline space, but there is no way to see that from this report.
