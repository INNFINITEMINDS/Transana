Clip Data Export

Transana's Clip Data Export routine creates a tab-delimited text file that allows you to look at information about Clips you've created in Transana with other programs such as SPSS and Excel.

If the Clip Data Export routine is called from an Episode, a listing of all Clips created from that Episode (regardless of what Collections those Clips are placed in,) is sent to the Filter Dialog, along with a list of all Keywords applied to that set of Clips. If the Clip Data Export routine is called from a Collection, a list of all Clips gathered together in that Collection (regardless of what Episodes they are from,) is sent to the Filter Dialog, along with a list of all Keywords applied to that set of Clips.

The first line of this text file is made up of column headings or variable names. Each subsequent line represents an individual Clip.

The first column of this text file is the name of the Collection where the Clip is stored. Nesting information for that collection is not provided due to lack of space.) The second column is the Clip Name, and the third column is the name of the media file used by the Clip. The fourth and fifth columns are the Clip Start and Clip Stop times in H:MM:SS.s format and the sixth column is the Clip length in seconds, accurate to the frame and listed in thousandths of a second. Subsequent columns are the coding data for the clips, with the Keyword Group and Keyword indicated in the first row of the column, and with a 0 indicating that the specified keyword was not applied to the specified clip and a 1 indicating that the keyword was applied to the clip.

A sample file (put into table format to make it easier to read) might look something like this:

Collection Name Clip Name Media File Clip Start Clip End Clip Length (seconds) Demo : Flip Demo : Symmetrical
Symmetry Carpet E:/Video/Demo/Demo.mpg 0:03:53.0 0:04:08.8 15.7280 0 1
Symmetry Flip Up, Flip Down E:/Video/Demo/Demo.mpg 0:00:00.0 0:00:20.7 20.6540 1 0
Symmetry House E:/Video/Demo/Demo.mpg 0:03:34.4 0:03:53.0 18.6350 0 1
Symmetry Up Down Flip E:/Video/Demo/Demo.mpg 0:00:31.1 0:01:05.5 34.4240 1 0
