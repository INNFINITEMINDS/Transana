Document Window

The Document Window is where you can view and edit your Documents, Transcripts, Quotes, and Clips.

The Document Window is like a standard word processor for Documents. It can display multiple Documents on different tabs, and can even display multiple Transcripts within adjustable panes within a single tab.

The Document Window has some special features build in to make the act of transcribing media files and working with media files linked to Transcripts easier. For example, you can add time codes, which link a location in the media file to a position in the Transcript. Once this is done, the highlight in the Transcript will move as the video is played, highlighting the appropriate section of the Transcript (if the Transcript is not in "Edit Mode.")

The Document Window toolbar includes speed buttons for inserting symbols commonly used in Jeffersonian Transcript Annotation. These symbols are , which indicates rising intonation, , which indicates falling intonation, , which signifies an audible intake of breath, and , which signifies a whisper or quieted speech. (See Transcript Notation in Transana's Help Menu for more information about Jeffersonian Transcription Symbols.)

The next set of buttons are the Show/Hide Time Codes toggle, denoted by the time code icon , and the Show/Hide Time Code Values, denoted by the  icon. The first of these buttons controls whether the time code symbol is displayed in Transcripts, while the second one controls whether the time code value, in human-readable form, is displayed. (Please note that it is not a good idea to edit the time code value text. This will not affect the underlying time code, but may cause display problems when Transana tries to hide this automatically-generated text.)

The Edit Mode/Read-Only toggle, denoted by the  icon, is next. Please note that Transana starts up in "read-only" mode, where Documents and Transcripts are not editable and the highlight selection in a Transcript moves with the media file as it plays. Pressing this button toggles you into the "edit mode," where you can edit Documents and transcribe and insert time codes into Transcripts. Having the Transcript highlight track the media file position is very useful when you are watching or listening to your media file, but when you are editing a Transcript, you don't want the cursor moving around in the Transcript when you play video, as this is very disruptive to editing.

The next set of buttons in the Document Window toolbar are: the Create Quick Quotes and Clips button , which allows you to create Quick Quote and Quick Clips if you have a Keyword selected in the Database Tree; the Keyword button , which allows you to edit the Keyword List associated with the currently-loaded Document, Episode, Quote, or Clip; and the Save button , which saves the current Document or Transcript.

The Propagate Changes button is next, as indicated by , which allows you to push changes in a Document out to the appropriate Quote, and changes in an Episode Transcript out to the appropriate Clips, or to push changes in a Quote out to exact copies of that quote and changes in a Clip Transcript out to exact copies of that Clip.

The Match Selection in Other Transcripts button, shown by the  icon, causes all Transcript selections to match the one in the selected Transcript pane when multiple Transcripts are open at the same time. The Play Multiple Transcript Selection button, shown by the  icon, causes the current selections in multiple-Transcript clips to be played as a way of previewing the Multiple Transcript Clip that would be created from the current set of selections.

If you have time codes in a Transcript, you can right-click the Transcript to play the section of the associated media file(s) between the nearest time codes where the cursor is currently placed. You can also highlight a larger section of the Transcript if you want to play a portion that crosses time codes. In addition, if you would like to play from the current position in the Transcript to the end of the loaded Episode or Clip, you can hold down the Ctrl key and right-click the Transcript on Windows or the Ctrl and "Command" key and click the Transcript on the Mac.

During transcription of Episodes, you may find it easiest to use Ctrl-S to start and stop video playback. This causes the video to be played in Transcription Mode, which causes an auto-rewind to be performed when play is initiated. (You can set the amount of the rewind, as well as the playback speed, using the Transcriber Settings tab on the Program Settings dialog.) In addition, Ctrl-D will pause and restart the video playback without auto-rewinding, Ctrl-A will rewind the video by 10 seconds, and Ctrl-F will fast-forward the video by 10 seconds. Ctrl-N will cause the next time-coded segment to be played, and Ctrl-P will cause the previous time-coded segment to be played. Finally, Ctrl-T will cause a time code for the current video position (as indicated in the Visualization Window) to be inserted at the current cursor position in the Transcript.
