The Sample Video

This tutorial describes the transcription and analysis of a sample video file, Demo.mpg. This file is available on the Downloads page on the Transana Web Site (http://www.transana.org). If you have not already done so, you should download this video now and store it on your hard drive in a location where you will be able to find it. For the sake of this tutorial, we will assume you have stored it as C:\Program Files\Transana 2\Sample\Demo.mpg if you are using Windows or in HOME/Movies/Sample/Demo.mpg if you are using Macintosh OS/X.

The sample video shows a second grade class learning geometry through a quilt design and construction activity. In this tutorial, you transcribe the short video and analyze it to explore the students' emerging understanding of symmetry.
