Clip Merge

Transana can consolidate two adjacent Clips into a single Clip. To initiate this process, right-click a Clip and choose "Merge Clips" from the popup menu. Transana displays the Clip's information, along with a list of Clips that either immediately preceed or immediately follow that Clip.

Please note that Clips must share a boundary time code to be considered adjacent, ending at the current Clip's start or starting at the current Clip's end. Clips also must be drawn from the same set of Episode Transcripts and share the same included media files.

Select a Clip from the "Clip to Merge" field, and the form changes to reflect the merged data. The Clip start or stop time changes, the Clip transcript(s) are combined, and the Clip keywords are merged. The Clip id is not automatically changed, but you can change it if you want to. If one of the merged Clips was serving as a Keyword Example, the merged Clip is substituted appropriately.

You are able to edit the merged Clips as you like. Then press "OK" to merge the Clips, "Cancel" to reject the merge, or select a different Clip to merge with the original selection.
