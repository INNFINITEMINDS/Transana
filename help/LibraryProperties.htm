Library Properties

A Library is simply a place that holds original source data that are in the form of Documents for text data and Episodes for media files. Libraries allow you to organize your source text and media data into categories should you wish to do so.

To add a Library in Transana, right-click (Ctrl-click on the Mac) the "Library" node in the Data window and choose "Add Library." You will see the dialog box shown above. The only required information is a unique "Library ID." "Owner" can be used to indicate the person who owns or is in some other way responsible for the Documents and Episodes in a Library, and "Comment" can be used in any way you wish. The "Default Keyword Group" is the Keyword Group that will be displayed by default when Documents and Episodes are added to this Library.
