Media Window

The Media Window allows you to view one or more media files. You can play or pause the video or audio, and you can change the position of the media file (though not very precisely) using the slider at the bottom of the window.

In the lower right corner of the Media Window, the   button allows you to capture a screen shot from video files. (Not all video formats are supported.) Transana will use the Media Conversion Tool to create a JPG graphic of the current video frame. (Depending on the video format, this can be a frame or two off sometimes.) If that video file's current Transcript is in Transana's main interface and in Edit mode, the screen shot will also be inserted into the Transcript at the current cursor position.

You can adjust the automatic sizing of this window by going to the Options Menu, and selecting Media Size, or by dragging the edges of the window. When you resize the Media Window, the other three windows automatically adjust their size and position to accommodate the Media Window size you select if the Auto-Arrange option is selected.
