Notes Browser

The notes browser is a tool designed to help you explore your analytic notes. To open the notes browser, open the "Tools" menu and select "Notes Browser."

The notes browser shows all notes, divided into categories by the type of Transana data object they are associated with because notes are often used differently depending on what type of data object they are associated with.

If you select a note in the notes tree, that note is loaded in the note editor on the right side of the notes browser for editing. You can insert a time stamp, save a copy of the note in plain text format, print the note, or search for a word or phrase in the note text.

If you right-click a note in the notes tree, Transana shows you the location of that note in the data window's database tree. You can rename the note, you can edit the note's Properties, or you can load the clip the note is attached to, if the note is attached to a clip.

If you right-click one of the four root nodes (Series, Episode, Transcript, Collection, or Clip), you can request a notes report that shows the information and text about all notes associated with that type of Transana object. Right-clicking the Database root node allows you to select a notes report that can include all notes in your entire database.

If you select the Note Search tab, you can type in a word or phrase. Press the search button to the right of the search text box and the note search tree shows only those notes that contain the word or phrase you entered. If you choose a notes report from the note search tree, the resulting report contains only those notes that contain the search text you specified.
