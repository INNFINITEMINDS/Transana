Propagate Document and Episode Transcript Changes

You will most likely need to change existing Documents and Transcripts at some point during your analysis. However, when you create a Quote or Clip in Transana, you create a copy of the source Document or Transcript, so changes in your Document or Episode Transcript are not automatically reflected in your Quotes or Clips. (There are some analytic circumstances where you may want a Quote or Clip Transcript to differ from the original source Document or Transcript.) However, there are many instances where it would be nice to be able to make an edit once and have the changes show up in all of the appropriate places.

The "Propagate Document and Transcript Changes" function allows a researcher to synchronize Documents with Quotes and Episode Transcripts with Clip Transcripts when desired. When you make a change to a Document or Episode Transcript that you would like to propagate out to the relevant Qutoes or Clips, place your Document cursor within the edited segment and press the  button in the Document window toolbar.

(Please see Propagate Quote and Clip Changes for a mechanism for keeping copies of Quotes and Clips synchronized with each other without editing the source Document or Transcript.)

The "Review Quote Changes" or "Review Transcript Changes" screen will be displayed for each Quote or Clip created from the same Document or Transcript that overlaps the current cursor position. Press "Update" to replace the Quote or Clip Transcript with the new text. "Update All" will apply the text changes to all Quotes or Clips from the same source Document or Transcript that overlap the current cursor position. (Use this button with caution!) "Skip" will prevent changes from being made to the current Quote or Clip, while "Cancel" will stop the change propagation process and revert all Quotes or Clips back to their original form.

This process sometimes yields results that differ from what you might expect. When you look at the "Selected Items" tab in the data window, it shows you all of the Quotes or Clips that have been defined that include the current position of the current Document or Transcript. However, you cannot assume that Transana will (or should) update all of these items if you select "Propagate Changes." Instead, take a moment to review the "Change Propagation" report.

If you have multiple Transcripts for an Episode, only those Clips derived from the current Transcript will be updated. If you want to update Clips derived from another Episode Transcript, update the other Transcript and then select the propagate button.

If the time codes in a Transcript have changed, Transana may not be able to locate the exact starting or ending points for a Clip. If this occurs, the Clip will not be updated during change propagation. It is a good idea to avoid deleting time codes once you have started creating Clips from a Transcript.

If a Quote or Clip Transcript is locked by another user in multi-user Transana, that Quote or Clip Transcript will also not be updated. You will need to wait until the other user releases the record lock, then repeat the change propagation process.

NOTE: If you are in a Quote or Clip when you notice the need to change your text, right-click the Quote or Clip in the Data window's database tree and choose "Locate Quote in Document" or "Locate Clip in Episode." Edit the original Document or Episode Transcript. Then press the  button to propagate the changes out to the appropriate items.
