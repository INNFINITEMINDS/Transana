Series Properties

To add a Series in Transana, right-click (Ctrl-click on the Mac) the "Series" node in the Data window and choose "Add Series." You will see the dialog box shown above. The only required information is a unique "Series ID." "Owner" can be used to indicate the person who owns or is in some other way responsible for the video, and "Title/Comment" can be used in any way you wish. The "Default Keyword Group" is the keyword group that will be displayed by default when Episodes are added to this Series.
