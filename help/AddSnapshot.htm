Snapshot Properties

Please note that the Basic version of Transana does not support Snapshots. To use this feature, you must be using the Professionsal, Computer Lab, or Multiuser version of Transana.

To add a Snapshot in Transana, you can right-click (Ctrl-click on the Mac) on the collection where you want to place the snapshot and select "Add Snapshot" from the popup menu. Doing this will add the snapshot to the end of the list in the collection.

If you want to place the new snapshot in a particular place in the list of items in a collection, you can right-click a clip or snapshot and choose "Add Snapshot,". If you do this, the new snapshot will be added above the selected item.

You will then see the Add Snapshot dialog box.

Press the Browse button to select an image file from the computer's disk, or press the "Capture Snapshot for Coding" button right next to it to capture a still image from the currently-loaded video file. (If you don't have an episode or clip loaded in Transana's main interface, this option is not visible.)

Please note that Transana does not import the image file into the database, but just stores the file name. If you move or delete the image file after creating a Snapshot, that Snapshot will not load properly. If you capture a snapshot from your video, you should avoid using the default Snapshot name to avoid accidentally deleting the image file created in this process should you take another snapshot from the same video file.

The only required fields are the "Snapshot ID," which must be unique within the parent collection, and the Image Filename. If you would like to link your snapshot to a particular spot in a media file, you can use the "Series ID", "Episode ID", "Transcription ID", "Episode Position" and "Duration" fields. "Comment" can be used in any way you wish.

You can apply coding to an entire still image by selecting a keyword group and keyword on this form, then clicking the  button. This allows you to find this snapshot later when doing keyword searches. The  button allows you to delete keywords from the snapshot keyword list, and the  button allows you to access Keyword Management. The "Collection ID" field is maintained by Transana and cannot be edited. (You can also code part of an image in the Snapshot Window.)

One final note: Unlike other branches of the database tree, items within a collection have an "order" that can be altered by the user. (This is important for "Play All Clips" and in the Collection Report.) If you choose "Add Snapshot", your new snapshot will be added to the end of the list of items within the collection. However, if you choose "Add Snapshot" on an existing clip or snapshot, the new snapshot will be inserted in the list above the item you selected. Finally, the order of items within a collection can be altered using cut-and-paste or drag-and-drop within the collection.
