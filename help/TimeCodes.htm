Time Codes

Time codes are marks that you can put into the text of your transcripts that link the media file and the transcript. They are very useful for synchronizing a transcript with the media file and are essential for defining the start and end points for analytically interesting clips within a larger media file.

You can right click on a portion of the transcript to play that section in the media file. If you play the media file from the video window or visualization window, Transana will highlight the appropriate section of the transcript as you go along, (as long as you are not in Edit Mode.)

To insert a time code, take the following steps:

1. Make sure Transana is in edit mode (by pressing the   button if necessary) so that clicking in the transcript to position the cursor doesn't move the media file position as it does when the transcript is in read-only mode.

2. With the program focus in the transcript window, use Ctrl-S (play/pause with auto-rewind), Ctrl-D (play/pause with no auto-rewind), Ctrl-A (rewind 10 seconds), and Ctrl-F (fast-forward 10 seconds) to position the media file at exactly the spot you want. For example, you may want a time code where the speaker changes, so you would pause right after the first speaker has finished talking and right before the second speaker begins to speak.

3. Position the transcript cursor in the correct spot in the transcript with a left-click. The location in the transcript should match the position where you paused in the media file. For example, you would place the transcript cursor after the sentence you just heard from the first speaker in your media file and before the sentence you are about to hear from the second speaker by clicking at the start of the second speaker's statement in the transcript.

4. Either press Ctrl-T in the transcript window or click the "Current" button in the visualization window to insert the time code for the current media file position in the currently selected position in the transcript.

For more information about inserting time codes, see Adding Time Codes and Advanced Time Codes in the Transana Tutorial.
