Collection Properties

To add a Collection in Transana, right-click (Ctrl-click on the Mac) the "Collections" node in the Data window and choose "Add Collection." You will see the dialog box shown above. The only required information is a "Collection ID" that is unique in the parent object. "Owner" can be used to indicate the person who created or is in some other way responsible for the Collection, and "Title/Comment" can be used in any way you wish. The "Default Keyword Group" is the keyword group that will be displayed by default when Clips are added to this Collection.

Collections can also be nested within other Collections. This allows Transana to represent more complex analytical structures and relationships within the Data Tree Display. Simply right-click (Ctrl-click on the Mac) a Collection and select the "Add Nested Collection" option.
