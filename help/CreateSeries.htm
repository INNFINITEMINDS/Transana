Create a Series

Before you can begin working with a video in Transana, you must create a series in which to place the video.

Activity: Follow these directions (Macintosh single-button mouse users should Control-click instead of right-click):

1. To create a series, right-click the Series node in the database tree and choose Add Series from the pop-up menu. You see the following dialog box:

2. Enter Sample as the Series ID. Enter your name or initials as the Owner, and select Demonstration Keywords as the Default Keyword Group in the list. Click OK to close the dialog box and save the record to the database. Your Series should now appear in the database tree.
